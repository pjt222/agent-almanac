# Vishnu Bhaga

存而養所運者——錨已驗之知，於擾下保一致，護功之模免於不必之變。

## 用時

- 運之法為範蔓或早優所擾之危
- 脈絡漂欲以陳假覆已驗之知
- 多並關生壓欲變宜恆者
- `shiva-bhaga` 解後——所存於重構中需主護
- 長會將失早已驗決於脈絡壓
- 對現行正功之系作變前

## 入

- **必要**：當前運之態或已驗欲存之知（隱可得）
- **可選**：穩定之特威（如「範蔓」「脈絡壓將至」）
- **可選**：MEMORY.md 與項目文以接地（以 `Read`）

## 法

### 第一步：盤點所運

護任何之前，識當前功且已驗者。

```
Preservation Inventory:
+---------------------+---------------------------+------------------------+
| Category            | Verification Method       | Anchoring Action       |
+---------------------+---------------------------+------------------------+
| Verified Facts      | Confirmed via tool use    | Record source and      |
|                     | (file reads, test runs,   | timestamp; do not      |
|                     | API responses)            | re-derive              |
+---------------------+---------------------------+------------------------+
| Working Code        | Tests pass, behavior      | Do not refactor unless |
|                     | confirmed, user approved  | explicitly requested   |
+---------------------+---------------------------+------------------------+
| User Requirements   | Explicitly stated by      | Quote directly; do not |
|                     | the user in this session  | paraphrase or infer    |
+---------------------+---------------------------+------------------------+
| Agreed Decisions    | Decisions made and        | Reference the decision |
|                     | confirmed during this     | point; do not revisit  |
|                     | session                   | without new evidence   |
+---------------------+---------------------------+------------------------+
| Environmental State | File paths, configs,      | Verify before assuming |
|                     | tool availability         | unchanged              |
+---------------------+---------------------------+------------------------+
```

1. 各類列當前已驗且運之具項
2. 注驗法——何以知此真？
3. 無驗之項不得護——其為假設（或需 `shiva-bhaga`）

得：具盤含已驗運之元與其證基。

敗則：盤稀者——少已驗——其本為寶信。先行 `heal` 以重接地，勿護未驗假設。

### 第二步：識擾源

名威穩態之力。

1. **範蔓**：任正擴逾所協否？
2. **脈絡漂**：早事正被近（或誤）思覆否？
3. **優壓**：欲改足運者否？
4. **外變**：境變否（文改、具不可得）？
5. **壓險**：對話將至脈絡限，早決或失否？

每源評：實威抑預險？

得：名擾源並評嚴重（活威 對 預險）。

敗則：無顯擾源者，存或不需——考 `brahma-bhaga`（創）或續執之宜否。

### 第三步：錨穩態

對所識威施特技以護所運。

1. **記憶之錨**：脈絡漂下危之要事，明重述：
   - 「立事：[X]，於對話 [點] 以 [法] 驗」
   - 持記可得者，書恆事入 MEMORY.md
2. **範界之執**：範蔓者，重述所協範：
   - 「協範：[原請]。當前工於此界內/外。」
3. **變之抗**：優壓下之運碼：
   - 「此件運且試。除非用者請，無變。」
4. **態之拍**：壓險者，立心點：
   - 結：何已畢、何餘、要決何
5. **境之驗**：外變者，前進前重察：
   - 重讀要文勿賴早讀

得：每識威有特錨應。穩態明護。

敗則：錨覺過——皆等護——分先後。何一物必不變？先護之。

### 第四步：行中養

存非被——後工中需續注。

1. 每行前察：「此威盤中物否？」
2. 若是，求別法達標而不擾穩態
3. 擾不可避者，明承且更盤
4. 周期重驗存項——尤於複操後
5. 任畢時，確存項仍完

得：運態於當前任完存。變唯於需處作而不擾運件。

敗則：存項誤變者，立評傷。變破物者，回。變中性者，更盤。勿留盤陳。

- [ ] 擾源已識且評
- [ ] 錨行已施於每實威
- [ ] 範界於任中守
- [ ] 存項畢後已重驗

## 陷

- **以假設為事而存**：唯已驗之知值護。未驗假設裝為事生假穩
- **過存**：皆等護阻必變。存必選——護所運，釋所不
- **被存**：假物自穩無主驗。脈絡漂常；存需續注
- **抗合法之變**：以存為避必修之托。用者請對運件之變者，此覆存
- **盤陳**：新信至而盤未更。盤須反當前實，非立時態

## 參

- `shiva-bhaga` — 毀先於存；解後所存乃 Vishnu 所養
- `brahma-bhaga` — 創建於所存基；新模出於穩地
- `heal` — 子系統評揭真功與表穩之異
- `observe` — 持中察於漂威穩前察之
- `awareness` — 境察（Cooper 色碼）直映擾察
