# 整合格式塔

由 `expand-awareness` 所生之全景覺知成一貫之全——非折衷之均，非擇最勝之域，乃辨獨視角所不能生之湧現之型。

## 用時

- `expand-awareness` 已現多域之原始覺知，諸察須成一統之見
- 多域視角俱備，而無單一能盡容其證
- 一問已從數角度析之，諸分析須超於列表
- 「合此諸觀為何意」之問無顯答
- 每整合皆墮為「擇最勝之域」而非生新
- `express-insight` 之前，須先有已成格式塔為入

## 入

- **必要**：`expand-awareness` 所生之多域觀察（或等同之全景覺知）
- **可選**：觸發此多域掃描之原問
- **可選**：格式塔所須守之既知約束
- **可選**：先前失敗整合之嘗試（墮為單域答者）

## 法

### 第一步：辨張力

於全景覺知所識各域對偶，述其關係。關係之三：張力（相悖）、共鳴（異角以固）、正交（無關之涉）。

用張力—共鳴對應圖：

```
Tension-Resonance Map
+-------------------+-------------------+-------------------------------+
| Domain Pair       | Relationship      | Detail                        |
+-------------------+-------------------+-------------------------------+
| A vs B            | tension /         |                               |
|                   | resonance /       |                               |
|                   | orthogonal        |                               |
|   Evidence:       |                   | What specifically disagrees,  |
|                   |                   | reinforces, or is unrelated?  |
|   Implication:    |                   | What does this relationship   |
|                   |                   | suggest for the whole?        |
+-------------------+-------------------+-------------------------------+
| A vs C            | ...               | ...                           |
+-------------------+-------------------+-------------------------------+
| B vs C            | ...               | ...                           |
+-------------------+-------------------+-------------------------------+
```

每對域填一行。N 域則有 N(N-1)/2 對。逾十行則先聚相關之域，再於群間映之。

以張力為先——含最豐整合之息。共鳴乃證；正交可置；唯張力需解，而格式塔出於解法之間。

**得：**張力—共鳴圖已全，每對皆有關係並具體證據。至少辨一真張力——若無張力，諸域或異未足以生湧現。

**敗則：**若諸對皆顯共鳴，諸域乃表面之合。當深察：何處因異故而同？同因異故乃隱之張力。若無關係可述，則 `expand-awareness` 之覺知或淺。返之，深域察再嘗整合。

### 第二步：尋其像

格式塔心理學中，像出於底。底乃第一步之張力—共鳴圖，像乃以最少矛盾統最多域之主型。

1. 察圖中之聚：何群之域相共鳴？此聚暗示候選之像
2. 於每候選問：「何視角能容最多觀察？」
3. 像非折衷（弱諸域以合），亦非擇（取最強者）。乃重設脈絡之新框
4. 試：以一句述候選。似屬某入域乎？若是，尚非格式塔——乃域答著偽裝
5. 特察張力：真像常居相悖諸域之間，而非任一域之位

像之將現之兆：
- 數張力於同一重框下並解
- 似悖之域察成同一現象互補之面
- 像解諸域各所見之故，含其所以悖

**得：**一二候選以單句述之。候選重設域察之脈絡，非擇取之。候選至少容圖中諸大張力。

**敗則：**若無像現，整合或早熟。二補救之徑：（甲）返 `expand-awareness` 添缺之域——或關鍵視角失，則像不能成；（乙）與張力共處，不強求解——某格式塔須孕而非勉。記當前之態，後歸再嘗。

### 第三步：驗全

第二步之候選須過三驗乃取。

**驗甲——張力之賬**：歷第一步每張力。格式塔解之乎，重框之乎，明承為不可化之取捨乎？未解張力示格式塔早熟。

**驗乙——單域之源**：此見可出自單域乎？若域專家頷首曰「吾儕已知」，則格式塔退為域答。真格式塔令諸域皆奇——各識其貢而不識其全。

**驗丙——輪察之貫**：心以每域視角次第察格式塔。形得保乎，抑因察域異而異？固之格式塔於任角皆一見；脆者因輪察而改義。

評：
- 三驗皆過：進第四步
- 驗甲敗：格式塔未全——返第二步，以未解張力為添束
- 驗乙敗：非湧現——返第二步，明除單域之框
- 驗丙敗：非一貫——或二見偽為一。分之，各半獨驗

**得：**候選格式塔過三驗，或敗態明識，導返第二步。

**敗則：**若數返皆敗，或諸域本不成自然格式塔。非每多域觀察皆生湧現——有時誠答乃結構域觀列，附其張力圖。交張力—共鳴圖為果，勿強偽一。

### 第四步：命其見

以一句述格式塔，域專家居其域不能獨書之。此句為交付之果。

1. 書此句。宜：
   - 具以可行或可否證
   - 廣以容諸貢之域
   - 令至少二入域奇
   - 無任一域之術語（或明重設脈絡之術語）
2. 以第三步之三準一驗
3. 可添一段擴，述格式塔自域貢如何湧現——此乃來歷，非見本身
4. 記貢之域、關鍵之張力、像—底之關係——此後設支援後整合之嘗

命之見與來歷，為 `express-insight` 傳達之入。

**得：**一句捕格式塔，附短來歷段。句過「無單域」之驗。任一貢域之行者讀之，識其貢而知獨不能至此句。

**敗則：**若句屢墮域語，試否證之驗：述格式塔非何。「此非安全之薦，非效能之優，非架構之式——乃[格式塔]。」否證清域框，予湧現之表達空間。

## 驗

- [ ] 諸域對偶之張力—共鳴圖已全，附具體證據
- [ ] 至少辨一真張力（非僅重點之異）
- [ ] 候選格式塔以重框述之，非折衷非擇
- [ ] 驗甲過：諸大張力皆解、重框、或承
- [ ] 驗乙過：無單域能獨生此見
- [ ] 驗丙過：格式塔於每域視角下保其形
- [ ] 終見以一句並來歷述之

## 陷

- **均之**：弱諸域之位至表面合。此生糜，非格式塔。若整合覺乏味，乃均之
- **立王**：取最強域之答，以多域之語飾之。驗乙捕之——若一域專家頷首不奇，乃立王
- **早閉**：受首候選而未以張力驗之。初現之像常最顯，而非最整合
- **強合**：於諸域真正交時仍執格式塔之存。正交域生結構列，非格式塔——此亦為正果
- **術語混**：集多域術語為句，似整合而實無義。終句每術語宜獨具義

## 參

- `expand-awareness` — 生此技所整合之原全景覺知；恆先於 integrate-gestalt
- `express-insight` — 將已成格式塔傳於受眾；恆繼於 integrate-gestalt
- `build-coherence` — 以結構評估於競項間擇；integrate-gestalt 生新全，非於既項擇
- `brahma-bhaga` — 自虛無而創；integrate-gestalt 自盈多（諸充視角）而創
- `meditate` — 清先脈絡以淨察；於 expand-awareness 之前用，而彼先於此技
- `coordinate-reasoning` — 管多路評估之資訊流；當格式塔涉調多推理線時互補
