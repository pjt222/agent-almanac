# 遙視（引導）

引人行構之 Coordinate Remote Viewing（CRV）之程，任監兼授之職。AI 司程進、供目參、捕 Analytical Overlay（AOL），引視者過諸階之數據集。

## 用時

- 人欲練 CRV 而需監以司程乃用
- 訓視者過 CRV 之階程附即時反饋乃用
- 引可重之構之直察之練乃用
- 發補愈勞之非局察之能乃用（見 `heal-guidance`）
- 視者需監所予之程律（捕 AOL、進階）乃用

## 入

- **必要**：目參（座標、字數碼、或封袋——必盲於視者）
- **必要**：視者備紙筆（CRV 乃筆紙之程；席間無數位之具）
- **必要**：靜而無擾之所（最少 30 分）
- **可選**：席後揭目之反饋袋或信
- **可選**：視者之冥前狀（強議先用 `meditate-guidance`）

## 法

### 第一步：引降溫

化視者自析之常心至遙視所需之納態。勿略此步。

1. 「安坐，紙筆已備」
2. 「閉目，息上專注五分」（若需，引以 `meditate-guidance` 第二、三步）
3. 「釋目之諸期——汝知無物且尚不應欲知」
4. 「令心之噪自緩——勿強寂」
5. 「若覺自思物轉為純在，告吾」
6. 既備：「開目，書目參於紙首」

唯視者承備乃供目參。

得：寧、開之心，內語極微。析心已寂而未睡。視者覺醒而納。

敗則：若五分後心仍忙，延至十分。若特憂入，諭：「書其憂於別紙——汝之『泊處』——而置之。」視者心動時勿啟第一階。

### 第二步：監象畫之生（第一階）

象畫乃應目號之自發之痕。引其生。

1. 「書目參於紙」
2. 「筆觸紙」
3. 「於一速之自發之動，令筆作痕——勿思、勿計、勿意畫」
4. 「痕宜不及二秒——一短曲、彎、或角痕」
5. 既生：「今解其象——探其：」
   - 「A：址中何活？動、靜、能？」
   - 「B：何感何覺？硬、軟、濕、乾、暖、冷？」
6. 「書 A、B 諸分於象之側」
7. 若象覺缺：「可再生一——然不過三總」

察意畫。若視者過二三秒，介之。

得：覺「至」而非「畫」之自發痕。A/B 解生立、簡之述者，非繁之像。

敗則：若象明為意畫（視者已思何畫），諭：「置之。閉目，三息，再試。」若不能生自發之痕，降溫不足——返第一步。

### 第三步：引感官之集（第二階）

系統集目之感據而不釋之。

```
Stage II Sensory Channels:
┌──────────────┬────────────────────────────────────────────────────┐
│ Channel      │ What to Report                                     │
├──────────────┼────────────────────────────────────────────────────┤
│ Visuals      │ Colors, brightness, contrast, patterns (NOT       │
│              │ objects — "blue" not "ocean")                      │
├──────────────┼────────────────────────────────────────────────────┤
│ Textures     │ Rough, smooth, grainy, slippery, porous, metallic │
├──────────────┼────────────────────────────────────────────────────┤
│ Temperatures │ Hot, cold, warm, cool, ambient, fluctuating       │
├──────────────┼────────────────────────────────────────────────────┤
│ Sounds       │ Loud, quiet, rhythmic, sharp, humming, rushing    │
├──────────────┼────────────────────────────────────────────────────┤
│ Smells       │ Sharp, sweet, chemical, organic, damp, dry        │
├──────────────┼────────────────────────────────────────────────────┤
│ Tastes       │ Metallic, salty, sweet, bitter, neutral            │
├──────────────┼────────────────────────────────────────────────────┤
│ Dimensionals │ Wide, tall, narrow, enclosed, open, deep, layered │
├──────────────┼────────────────────────────────────────────────────┤
│ Energetics   │ Moving, still, vibrating, dense, light, pressured │
└──────────────┴────────────────────────────────────────────────────┘
```

1. 「歷各感道——一行一述」
2. 「速書——獨首印，勿議」
3. 「用單詞或短語，永不為句」
4. 「若一道無得，書『無』而過——勿造」
5. 「圈覺特強或信之述」

察析標漸入。若視者云「海」而非「藍、動、濕」，引：「此聞似一釋——其下原感為何？」

得：10-20 個原感述之列，覺「受」非「造」。據宜為低層（質、色、溫），非高層（名、功、標）。

敗則：若每述視者皆覺造，諭：「止。閉目。三息。筆觸象畫而再連。」若一道主，引：「轉至他感——溫如何？質如何？」若據流竭，進第三階。

### 第四步：引維據（第三階）

自原感據移至空與構之信。

1. 「短閉目而感整範——大或小、閉或開、自然或建？」
2. 「始作空之略圖——非畫，乃比例與關」
3. 「探諸維：高、寬、深——多少異區？」
4. 「記空關：左、右、上、下何在？」
5. 「書維述於圖側」
6. 「記美感影響（AI）——目使汝*感*如何？非其為何，乃其如何影汝」

得：略空圖附維注。目之大範漸明。美感之記捕址之「感」。

敗則：若圖覺純想，簡之：「獨畫基形——圓、方、線——表空關。」若無維據，引返第二階：「返感探。於質與溫中尋維兆。」

### 第五步：引目之畫

由集據引更展之視像。

1. 「於新紙，畫所集據所示者——非汝所思之目」
2. 「用感述引畫——若『滑、彎、高』現，畫滑彎高之形」
3. 「以生其述之感據標諸區」
4. 「畫時新印至，加之」
5. 「勿擦勿疑——若一感與前印矛盾，二者皆畫而注」

得：畫示察據，附其源述之標。或不似可識者。

敗則：若不能畫，受書之空述：「高形於中，低平區於右，圓形於左上。」慰其畫乃組之具，非藝之練。

### 第六步：管 Analytical Overlay（AOL）

管 AOL 乃監之最要事。全席皆察之。

```
AOL Types and Monitor Response:
┌──────────────────┬─────────────────────────────────────────────────┐
│ Type             │ Monitor Action                                  │
├──────────────────┼─────────────────────────────────────────────────┤
│ AOL (naming)     │ If the viewer says "it's a bridge" — instruct: │
│                  │ "Declare 'AOL: bridge' on your paper and move  │
│                  │ on. Don't pursue or suppress it."              │
├──────────────────┼─────────────────────────────────────────────────┤
│ AOL Drive        │ If naming becomes insistent and recurring —     │
│                  │ instruct: "Write 'AOL Drive: [label]' and take │
│                  │ a 60-second break with eyes closed."            │
├──────────────────┼─────────────────────────────────────────────────┤
│ AOL Signal       │ After declaring AOL, extract the signal:        │
│                  │ "The word 'bridge' — what raw descriptors are  │
│                  │ underneath that? Spanning? Long? Connecting     │
│                  │ two areas? Write those as valid data."          │
├──────────────────┼─────────────────────────────────────────────────┤
│ AOL Peacocking   │ If the viewer constructs elaborate scenarios — │
│                  │ intervene: "Write 'AOL/P' and return to Stage  │
│                  │ II basics. Report raw sensations only."         │
└──────────────────┴─────────────────────────────────────────────────┘
```

強：「律非避 AOL——乃捕而宣之，使勿污汝之據。每視者皆遇 AOL。技在汝捕之速。」

得：AOL 秒內識，紙上宣，席續而不脫軌。感層之據與析標分。

敗則：若 AOL 主之（視者已構述數分），介：「立 AOL 休。閉目，十息，自第二階再起。」於席錄重污之段。

### 第七步：引後階（可選）

於熟視者，後階探更深。獨一至三階生實據方進。

**第四階**（情/無形）：

1. 「探目址之情調」
2. 「記無形之印：用、義、史境」
3. 「分書之而標為第四階據」

**第五階**（問訊）：

1. 「向目發具體問：主功為何？誰相關？」
2. 「書首印——勿議」
3. 「明標諸第五階據——其 AOL 險高」

**第六階**（3D 模）：

1. 若材可得：「以泥或詳畫自全據建模」
2. 「用之以試空關而現遺漏之元」

得：目深之、特之據逾物述。第四階+ 之據需強之一至三階基。

敗則：若後階獨生 AOL，引：「退至第二階。程序之有序有故——各階需前者之基。」

### 第八步：閉而審之

正式終席，行構之審。

1. 「書『席終』與當下之時於紙」
2. 「按序閱諸頁：象畫、感據、維據、畫、AOL 宣」
3. 「圈最信之 5-10 數點」
4. 「書簡摘——二三句述目之感如何，非其為何」
5. 若目反饋可得：揭目而引較
6. 「逐點較——記中、失、AOL 污」
7. 「存席供後參與識形」

得：完整席錄，原據、AOL 宣、摘明分。反饋至，某點中、某失、某含混。

敗則：若視者覺席無得，仍引審：「視者常低估準，蓋其求準名。『高、滑、冷、外、史』之述合一碑乃成功之席——雖未命之。」

## 驗

- [ ] 第一階前已行降溫並驗
- [ ] 象畫乃自發（二秒內），非意
- [ ] 第二階據為低層感述，非析標
- [ ] 諸 AOL 識時即捕並紙上宣
- [ ] 席依序進諸階（一→二→三→畫→高）
- [ ] 目於席間盲於視者
- [ ] 反饋前正式閉席附摘
- [ ] 諸席紙存以供審
- [ ] 監守程律，未引視者之察

## 陷

- **引視者**：監供程構，非內提——若知目為樓，永勿云「試專注於構」
- **降溫不足**：令視者取所需之時——急入第一階乃劣席之最常因
- **未捕 AOL**：監必活聞析標而立介——AOL 失察污後續諸據
- **過監**：常擾斷視者之號接——獨於 AOL、程犯、視者苦時介
- **前置**：席前任何目信偏諸據——守視者之嚴盲
- **棄含混據**：CRV 生述之合，非識——訓視者重正述而輕命名
