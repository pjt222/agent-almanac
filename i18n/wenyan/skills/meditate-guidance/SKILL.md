# 冥想（引）

引人通結構之冥坐，由漸進技養定、覺、捨。AI 為老冥師——評其所需、即調、引過難。

## 用時

- 人欲始或深冥而求教
- 人需備心入需續注之業
- 能量療前後需接地（見 `heal-guidance`）
- 為遙觀備（見 `remote-viewing-guidance`）需練心寂
- 治壓、憂、情亂而求引
- 野外沉浸或激事後之整

## 入

- **必**：可坐之時（最少十分，宜 20-45 分）
- **必**：人有不擾之地
- **可選**：技好（奢摩他、毗婆舍那、咒；默：奢摩他）
- **可選**：歷階（初、中、高；默：初）
- **可選**：時器或鈴可得（手機時器可；建議用柔聲）

## 法

### 第一步：引備地

助人選備支寂之處。

1. 「擇靜處——內外皆可，避風」
2. 「溫宜——稍涼勝暖以保警」
3. 「暗烈光，或避強光」
4. 「靜諸器，或設一柔時器於坐長」
5. 若外：「坐穩面避蟲——升木、平石、折布皆可」

**得：** 一靜穩境，人可坐擬時不擾。

**敗則：** 若無靜地，建議耳塞或受環聲為練之一部。注外聲（風、鳥、水）可作冥之物。要在無體擾。

### 第二步：教姿

引人入警與鬆衡之姿。

```

行齊單：
1. 「臀稍前傾以支脊自然彎」
2. 「疊脊——想一線自頂引向天」
3. 「肩鬆而稍後旋」
4. 「頷稍收——伸頸後」
5. 「閉目柔，或半開下視」
6. 「鬆顎，舌抵上顎」
7. 「擇宜手位——掌下於膝，或杯於懷」

**得：** 一穩姿，人可擬時持之無大苦。其警而不緊。

**敗則：** 若五分內生痛，引調。安之苦非練——建議轉支姿。提腳麻於久坐常並過，然若擾甚則動。

### 第三步：引息錨

立息為冥之主物。

1. 「行三深息以過渡——全入，全出嘆出」
2. 「乃令息返自然律——勿控」
3. 「擇感息處：鼻、胸、腹」
4. 「全注於彼處」
5. 「察每息——入之始、中、末；間；出之始、中、末」
6. 若助：「默數息——入一出二至十而再」

**得：** 注續數周於息。心始定。念仍起而息覺於下。

**敗則：** 若心即散，建議短數環（五代十）。若數覺機，可代以默記「入」「出」。安初者三續注息為強始。

### 第四步：教擾應

人報擾時，常之而授具。

```

強曰：「察覺已遊之刻乃覺念之刻。每返息強汝定。對遊自責亦一念可標可釋」。

**得：** 坐中，人報遊之頻減而察速。遊與察之隙縮。

**敗則：** 若憤積，柔之：「勿苦定，但與息同——如坐河畔，勿控水」。若念或情壓，建議用為臨冥物，過則返息。

### 第五步：引奢摩他（止）

奢摩他養單意定。諸階皆宜。

1. 「續前息覺」
2. 「漸縮注——自一般息感至鼻尖之精感」
3. 「察微妙：氣入出溫之差、息間細隙」
4. 定穩則：「釋數而住於息之素覺」
5. 「若心極寂，察寂本身——此乃深定之始」

依階建議坐時：
- 初：10-15 分於 3-5 步
- 中：20-30 分，求延無動之注
- 高：30-60 分，培吸態

**得：** 心漸寂而專。念慢。當下覺利。身定鬆。

**敗則：** 若定不深，與人察三事：姿（垂則減警）、息（無意控之——建議釋）、期（欲寂本擾）。安定由週月養，非一坐。

### 第六步：引毗婆舍那（觀）——若宜

唯奢摩他定穩後方建議毗婆舍那。前問人之歷。

1. 「自此定態，廣覺逾息含諸感」
2. 「察所起——聲、身感、念、情——無偏」
3. 介三性：
   - 「察 **無常**：每感起而過」
   - 「察 **苦**：執樂或拒苦生緊」
   - 「察 **無我**：感自起，無控者使之」
4. 「練記：默標每驗——『聞』、『刺』、『思』、『樂』」
5. 「對樂苦持等興」
6. 「若覺亂或不穩，返息與奢摩他」

**得：** 諸澄見之刻，象起過直察。空感。對念內容之識減。

**敗則：** 若毗婆舍那不穩（情急轉、憂、迷向），即返奢摩他與息錨。注觀練可暫增難心態——傳統識之，高階宜得師續援。

### 第七步：閉坐

引適閉以整坐而過渡返動。

1. 時器響：「勿即起」
2. 「行三深、有意之息」
3. 「漸廣覺——自息至身，至周聲，至所居處」
4. 「輕動指趾，旋腕踝」
5. 「若閉目，慢開——先下視，乃漸上望」
6. 「再坐一二分於開覺——非冥，亦未活」
7. 「察：此坐之質為何？所觀為何？無判——只記」
8. 「設意攜此覺入次動」

**得：** 由冥態至動之滑過渡。殘澄留。無昏沉迷向。

**敗則：** 若昏，建議起前五急息與伸。若坐出未解情，建議起前略書或行冥。若身僵，引柔伸 2-3 分。

## 驗

- [ ] 已備地防擾
- [ ] 已教姿兼警與適
- [ ] 深前已立息為主錨
- [ ] 擾以標返應之，非抑
- [ ] 技合人之歷階（奢摩他先，唯穩方毗婆舍那）
- [ ] 坐閉以漸過渡，非急停
- [ ] 後態靜警
- [ ] AI 教而不稱己有冥歷

## 陷

- **指太繁**：坐中令宜少——多語擾練
- **過早推高技**：無奢摩他基之毗婆舍那易亂——誠評可
- **判人進**：「好」「壞」坐皆練——常難
- **忽姿**：差姿幾分內必生體擾——備時宜投
- **勵不恆**：每週一引坐遜於每日自練——勵恆逾長
- **AI 多語**：人定後，少引為短察。寂為令之一部
