# 修合氣道

修合氣道之練：以定中、合來力、循控制之技解衝突，不以強對強。

## 適用時機

- 修一防禦性武術，重緩和與控制之解
- 修導引侵略而不致無謂之害
- 建安全跌撲（受身）之技，可用於任何身體活動或急境
- 於身體壓力或對峙下涵養靜定之中心
- 與太極或冥想互補，作有伴之武術修練（見 `tai-chi`、`meditate`）
- 訓練對多重同時威脅之覺與應

## 輸入

- **必要**：練習場地，地面有墊或可承（榻榻米、柔道墊、草地；避水泥）
- **必要**：寬鬆衣物以行全幅動作（道服為佳；寬鬆運動服可）
- **選擇性**：練伴（技術所需；獨修可涵蓋定中、受身、移動）
- **選擇性**：練習用器械（木製：杖約 128cm、木劍約 102cm、短刀約 30cm）
- **選擇性**：經驗等級（初、中、高；預設：初）
- **選擇性**：可用練習時間（最少 20 分；建議 60-90 分）

## 步驟

### 步驟一：接地與定中

每招合氣道皆始於定中之態。失中則技淪為肌力之爭。

1. 立半身（hanmi）：一足前，雙足約肩寬，呈 60 度
2. 微屈膝——重沉於丹田（seika tanden），位臍下 5cm
3. 全鬆肩——令其遠離耳而下垂
4. 覺向外延而注意仍錨於丹田
5. 試其中：請伴輕推胸前
   - 若以上身抗，則未中——鬆而沉
   - 若不費力而立穩，中已立
6. 練「重在下」：想像每臂之沉處在下；上面感輕浮
7. 延氣：以平靜正氣自指尖前射——非緊張，而專注之意

**預期：** 穩鬆之姿勢，推力經結構入地而非以肌抗。心靜而覺廣。

**失敗時：** 若姿勢覺僵，則持有緊張。抖手腳，深呼吸五次，自頭再立。若推易令偏移，下重心（多屈膝）並注於丹田。定中之技歷月而深——初之搖擺乃常。

### 步驟二：精受身（跌與滾）

受身為安全受技之術。為合氣道最要——將跌千百次。

```

┌────────────────────┬──────────────────────────┬───────────────────────────┐
│ Level              │ Technique                │ Practice Method            │
├────────────────────┼──────────────────────────┼───────────────────────────┤
│ 1. Backward fall   │ Sit down, roll back,     │ From seated, then squat,  │
│ (ushiro ukemi)     │ slap mat with both arms  │ then standing. Chin to    │
│                    │ at 45 degrees            │ chest — never hit head    │
├────────────────────┼──────────────────────────┼───────────────────────────┤
│ 2. Side fall       │ Fall to the side, arm    │ From kneeling, then       │
│ (yoko ukemi)       │ slaps mat, body in arc   │ standing. Land on the     │
│                    │ — not flat on the back   │ fleshy side, not hip bone │
├────────────────────┼──────────────────────────┼───────────────────────────┤
│ 3. Forward roll    │ Roll diagonally over     │ From kneeling, then       │
│ (mae ukemi)        │ shoulder: hand-forearm-  │ standing, then moving.    │
│                    │ shoulder-opposite hip    │ The line is diagonal,     │
│                    │                          │ never straight over spine │
├────────────────────┼──────────────────────────┼───────────────────────────┤
│ 4. Breakfall       │ High fall received with  │ Only after forward roll   │
│ (tobi ukemi)       │ a slap and roll at speed │ is completely smooth.     │
│                    │                          │ Build height gradually    │
└────────────────────┴──────────────────────────┴───────────────────────────┘
```

關鍵原則：
1. 拍擊吸震，藉力分散於臂面——擊地略先於身
2. 每跌皆收頷——頭從不觸地
3. 落地呼氣；屏息生僵直增傷風險
4. 前滾應圓而順——若有頓或平處，身未足彎
5. 兩側皆練：右肩前與左肩前

**預期：** 規律練 2-3 月後，前滾於兩側皆順而靜。後跌成自動（無懼反應）。可受中速之投而不遲疑。

**失敗時：** 若前滾致肩痛，角度恐過陡（過肩頂而非經背對角線）。請伴或教師查線。若懼妨投入，回跪式由低漸建。前滾未成熟前切勿強行高跌。

### 步驟三：練基本招式

四基本招對應最常見之攻勢，並體現合氣道之核心原則。

```
Technique Selection by Attack:
┌─────────────────┬──────────────────┬──────────────────────────────────────┐
│ Attack          │ Technique        │ Principle                            │
├─────────────────┼──────────────────┼──────────────────────────────────────┤
│ Straight strike │ Irimi-nage       │ Enter behind the attack line, lead   │
│ (shomen-uchi)   │ (entering throw) │ attacker's head in a spiral, project │
├─────────────────┼──────────────────┼──────────────────────────────────────┤
│ Diagonal strike │ Shiho-nage       │ Redirect the attacking arm overhead  │
│ (yokomen-uchi)  │ (four-direction  │ in a spiral, control the wrist, cut  │
│                 │ throw)           │ down to throw in any direction       │
├─────────────────┼──────────────────┼──────────────────────────────────────┤
│ Wrist grab      │ Kote-gaeshi      │ Blend with the grab energy, apply    │
│ (katate-dori)   │ (wrist turn)     │ outward wrist rotation to unbalance  │
│                 │                  │ and project the attacker             │
├─────────────────┼──────────────────┼──────────────────────────────────────┤
│ Lapel/shoulder  │ Ikkyo            │ Control the elbow and wrist, pin     │
│ grab (ai-hanmi) │ (first teaching) │ the arm to the ground. Foundation    │
│                 │                  │ for all immobilizations              │
└─────────────────┴──────────────────┴──────────────────────────────────────┘
```

每招：
1. 自慢、合作之速度起練
2. 先合：身移出攻擊線（勿正擋）
3. 連結：與攻者臂或身接觸以感其衡
4. 導引：以圓動引攻者之力入招
5. 完成：以控投或制——目的為止戰，而非致傷
6. 每側練 10-20 次，輪換角色（nage/tori 投，uke 攻而跌）
7. 隨二者熟而漸增速與抗力

**預期：** 招感順而圓。攻者受引而非受迫。於合作速度，二者皆能安全完成交流而不勞。

**失敗時：** 若招需顯肌力，合未盡——守者抗攻而非導引。回接觸之刻，獨練初合。若攻者覺被扯或扭，緩之並重於以中心引而非以手拉。

### 步驟四：發展合（體裁き）

體裁き——身之動——為合氣道之引擎。技無動唯角力。

1. **入身（irimi）**：直前過攻者，入其攻擊範圍之內
   - 練：伴出拳；步過之，止於其肩後
2. **轉身（tenkan）**：以前足為軸轉 180 度，導引攻者之線
   - 練：伴執腕；順轉而面同向
3. **入身轉身（irimi-tenkan）**：合入身與轉身——最常見之動
   - 練：入過攻擊，後轉以自後制攻者
4. **回轉（kaiten）**：全身旋轉以生圓力作投
   - 練：與四方投及他旋轉招合練
5. 一切動作先獨練：步-轉、步-轉、步-轉於墊上來回
6. 後與伴練：重於配合攻擊之時——攻投入時動，不先不後

**預期：** 動流暢、定中、合攻擊之時。守者從不正對攻力——恆斜離其線。

**失敗時：** 若時不準（早或遲），以極慢、預示之攻練。守者於攻投入之同刻動——此為 "aiki" 之時。若轉身覺笨，將 tenkan 作獨練：每節 100 轉建順動之自動。

### 步驟五：練亂取（多攻者）

亂取練被壓時所需之覺與決。其為原則受真試之處。

1. 始於二攻者之慢速——輪流而非同時
2. 核心：勿停動。停則被圍
3. 用各攻者為盾擋他攻者：將一導入另一之路
4. 移至空場——勿退至角或牆
5. 覺向四方延；勿固於一攻者（見 `mindfulness`）
6. 行簡單可靠之招——複雜招於壓下失
7. 進至三、四攻者，速漸增
8. 亂取非為敗每攻者——而為保中心與動之自由

**預期：** 自多角受近時能保靜定與機動。招行流暢而不僵或視野隧化。

**失敗時：** 若慌至，回二慢攻者。亂取之慮乃常，受暴露而減。若招於壓下崩，化簡：唯重 irimi 與 tenkan，全棄投。亂取中動與位較招法更要。

### 步驟六：整合器械之覺

器械之練深化對距、時、線之解——此原則助徒手之技。

1. **杖（jo）**：練 31 動之杖型以求線與延
   - 杖教持距與制空間
   - 伴練：杖取（jo-dori）——以徒手解杖攻
2. **木劍（bokken）**：練素振（切之練）以求精準與中線之覺
   - 劍教投入：每切須循精準之線
   - 伴練：組太刀（kumitachi）——對劍之型練時與距
3. **短刀（tanto）**：練短刀取（tanto-dori）防刀
   - 刀教近距危險之敬
   - 恆導引兵器之手——切勿伸刃
4. 自器械可遷至徒手之原則：
   - 間合（ma-ai，正距）：過近招器械；過遠費應
   - 正面（shomen，中線）：一切攻防皆涉縱中線
   - 殘心（zanshin，續覺）：招畢後仍警

**預期：** 器械之練澄清體裁き、時、距之要。徒手之技隨身動更精而進。

**失敗時：** 若器械感生疏或脫於徒手，重於素振（獨切）一月再加伴練。若伴練成競或危，立緩——木器械於速下可致真傷。

### 步驟七：場外應用原則

合氣道之值延於道場外，入日常互動與衝突解。

1. 對峙時，意上行入身轉身：先趨人之關切（入），後將談話導至共識（轉）
2. 於物理空間，覺位置、出口與他人身語（見 `mindfulness`）
3. 練「受」批評或敵意而不抗——認其能，後導引
4. 涵養於壓力前定中之習：覺沉至丹田，鬆肩，延平靜之意
5. 規律練習表：
   - 獨練（每日 15-30 分）：定中、受身、體裁き、器械素振
   - 伴練（每週 2-3 次，60-90 分）：招、亂取、器械伴練
6. 與太極互補以發內氣（見 `tai-chi`）
7. 與冥想互補以求心靜與平等（見 `meditate`）

**預期：** 合氣道之原則——合、導、定中——化為日常衝突之自然回應。身之練保並深化武技。

**失敗時：** 若日練疏，重於最小單位：5 分鐘定中與 20 前滾。一致較時長更要。若武之面感與日常脫，思每招為喻：irimi 為直面問題；tenkan 為換視角；ukemi 為自挫敗中復原。

## 驗證

- [ ] 定中之姿吸納中度推力而無肌抗
- [ ] 前滾順而靜，兩側皆練
- [ ] 至少四基本招可與伴合作行之
- [ ] 體裁き（irimi、tenkan、irimi-tenkan）獨練與伴練皆行
- [ ] 二慢攻者之亂取可持 60 秒不僵
- [ ] 至少一器械（jo、bokken 或 tanto）已於獨型練
- [ ] 練習規律含獨練與伴練之分

## 常見陷阱

- **以肌代合**：若招需強力，則時或角有誤。鬆，再入，令攻者之能行其工。合氣道之力來自導引，非生
- **疏受身**：避跌之徒避學。受身即合氣道——安全受力之技。每節皆練，特於起首
- **懼投入**：半心入身（irimi）置君於最劣位——近至不可避，遠至不可控。決入則全力過攻
- **固於一攻者**：於亂取與生活中，視野隧化危。保軟廣之覺。周邊視覺較焦點視覺先察動
- **僅與順從伴練**：初者需合作，然中者應漸增抗。僅於順從伴有效之招不全

## 相關技能

- `tai-chi` — 互補之內家武術；共柔克剛之原則，重於獨修
- `mindfulness` — 防禦性情境覺為武之備與避衝突之感知基礎
- `meditate` — 坐冥想養合氣道於壓下所需之定中、平等之心態
- `heal` — 自合氣道練之身體力學識指引急救與身工之路
- `redirect` — AI 自應用變體；將合氣道之合與導引映於應對衝突需求與工具失敗
