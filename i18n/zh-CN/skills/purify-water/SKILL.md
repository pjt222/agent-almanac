# 净化水源

使用野外可用的方法将野外水源净化为安全的饮用水。

## 适用场景

- 在没有处理过的水源的野外环境中需要饮用水
- 可用水源质量未知（溪流、河流、湖泊、池塘）
- 紧急求生场景中存在脱水风险
- 需要安全的水用于烹饪或伤口清洗

## 输入

- **必需**：水源（流动或静止）
- **必需**：容器（金属锅、水瓶或临时容器）
- **可选**：净化用品（化学净水片、过滤器、紫外线笔）
- **可选**：生火能力用于煮沸（参见 `make-fire`）
- **可选**：布料或天然过滤材料用于预过滤

## 步骤

### 第 1 步：评估和选择水源

不同水源的风险程度不同。选择最佳可用水源。

```
水源优先级排序（从最佳到最差）：
┌──────┬─────────────────────────┬────────────────────────────────────┐
│ 排序 │ 水源                     │ 说明                              │
├──────┼─────────────────────────┼────────────────────────────────────┤
│ 1    │ 泉水（源头处）           │ 污染最低；仍需处理                 │
│ 2    │ 快速流动的溪流           │ 流动水比静水病原体少               │
│      │ （人类活动区上游）       │                                    │
│ 3    │ 大河                     │ 稀释有帮助，但需注意上游的         │
│      │                         │ 农业/工业活动                      │
│ 4    │ 大湖                     │ 从开阔水面取水，不要在岸边取       │
│ 5    │ 小池塘或水坑             │ 病原体和寄生虫风险高               │
│ 6    │ 死水潭                   │ 最后选择；需要强力处理             │
└──────┴─────────────────────────┴────────────────────────────────────┘

警告信号（尽可能避开）：
- 附近有死亡动物
- 蓝绿藻水华（蓝绿色浮渣）
- 化学气味或油膜
- 位于矿区、农业区或居民区下游
- 周围没有植被（可能表明土壤有毒）
```

从水面下方取水（避免表面薄膜），远离岸边。

**预期结果：** 从最佳可用水源中获取清澈或略有浑浊的水，盛入干净的容器中。

**失败处理：** 如果只有劣质水源（死水、浑浊），继续但计划进行强力预过滤（第 2 步）并使用多种净化方法（双重保险）。如果找不到水源，寻找指示信号：山谷中的绿色植被、向下坡移动的动物踪迹、黎明/黄昏时的昆虫群，以及倾听流水声。

### 第 2 步：预过滤沉淀物

在净化前去除颗粒物。沉淀物会降低化学处理的效果并堵塞过滤器。

```
临时重力过滤器（在底部有孔的容器中分层）：

    ┌─────────────────────┐  ← 顶部开口：倒入水
    │  草 / 布            │  ← 粗过滤层
    │  细沙               │  ← 去除细颗粒
    │  木炭（碎末）        │  ← 吸附部分化学物质和异味
    │  碎石               │  ← 结构支撑
    │  草 / 布            │  ← 防止碎石掉落
    └────────┬────────────┘
             │
        过滤水滴出

材料：
- 容器：桦树皮卷筒、空心木头、剪开的塑料瓶、袜子
- 沙子：细的、干净的沙子（如有可能先冲洗）
- 木炭：来自之前的火堆（不是白灰——仅用木炭）
- 碎石：小石头，冲洗过
```

简单去除沉淀物可以用头巾、T恤或多层布过滤水。

**预期结果：** 明显变清的水，浑浊度降低。木炭层可去除部分异味和杂味。

**失败处理：** 如果过滤后水仍然非常浑浊，让水在容器中静置 30-60 分钟，然后小心倒出上层较清的水。重复静置或过滤过程。注意：预过滤不能使水安全饮用——它是为净化做准备。

**预期结果：** 根据可用工具明确决定使用哪种净化方法。

**失败处理：** 如果没有标准净化工具，煮沸是默认选择——只需要火和耐热容器。即使是单层金属水瓶也可以用来煮沸。在极端紧急情况下，可以用岩石凹陷处或绿竹段靠近火焰来加热水作为临时容器。

### 第 4 步：煮沸水

最可靠的野外净化方法。杀灭所有种类的病原体。

```
煮沸步骤：
1. 将水加热至滚沸（大气泡冲破水面）
2. 保持滚沸时间：
   - 海平面至2000米 / 6500英尺：1分钟
   - 2000-4000米 / 6500-13000英尺：3分钟
   - 4000米 / 13000英尺以上：5分钟
3. 离火
4. 盖上盖子冷却
5. 如果味道平淡，在两个容器之间反复倒水以增加含氧量

海拔调整：
  在高海拔水的沸点更低。
  在3000米 / 10000英尺处，水在约90°C / 194°F时沸腾。
  延长煮沸时间补偿较低的温度。

燃料估算：
  煮沸1升水大约需要15-20分钟的持续火焰，
  具体取决于容器、风力和起始温度。
```

**预期结果：** 水达到剧烈滚沸并保持适当时间。冷却后，水可安全饮用，无生物病原体。

**失败处理：** 如果无法维持滚沸（风大、火弱），延长煮沸时间。如果容器漏水或破裂，转用其他容器。如果没有金属容器，可以用木头、树皮或皮革容器加热石头来煮水：将石头在火中加热 20 分钟以上，然后用木棍或树枝钳入水容器中。避免使用河石（可能因内部含水而爆裂）。

### 第 5 步：化学处理

当煮沸不可行时使用，或作为辅助处理方法。

```
处理后，水应有轻微的氯/碘气味。
如果闻不到气味，追加原始剂量的一半并再等15分钟。

冷水/浑浊水调整：
- 温度低于5°C / 40°F：等待时间加倍
- 浑浊水：剂量加倍或先预过滤（推荐）
```

**预期结果：** 等待期后处理过的水有微弱的化学气味，表示消毒充分。水可安全饮用，无细菌和病毒；二氧化氯对寄生虫也有效。

**失败处理：** 如果药片过期（处理后无气味），使用双倍剂量或结合其他方法。如果味道难以接受，让水敞开放置 30 分钟使气体散发，或通过临时木炭过滤器改善口感。如果化学处理是唯一方法且怀疑有隐孢子虫（常见于牲畜附近），二氧化氯需等待完整的 4 小时，或结合过滤使用。

### 第 6 步：安全储存

净化水可能因脏容器或脏手而被再次污染。

```
安全储存方法：
1. 储存在干净的专用容器中（不要重复使用未净化水的容器）
2. 如果重复使用容器，先用少量净化水冲洗
3. 保持容器密封或遮盖
4. 标记或区分"生水"和"净化水"容器
   （例如在净化水瓶的伞绳把手上打一个结）
5. 避免用手伸入容器——倒出来用，不要用手舀
6. 在温暖天气中，24小时内饮用
7. 储存超过24小时的水需重新处理

水合计划：
- 最低需求：每天2升 / 0.5加仑（静坐、凉爽天气）
- 活动量大：每天4-6升 / 1-1.5加仑（徒步、炎热天气）
- 规划净化能力以满足每日需求
```

**预期结果：** 净化水在干净、密封的容器中保持安全。建立了避免生水和处理水交叉污染的系统。

**失败处理：** 如果容器有限，指定一个作为"生水"（仅用于取水）另一个作为"净水"（仅放净化水）。明显标记区分。如果怀疑再次污染，饮用前重新处理水。

- [ ] 浑浊水在净化前进行了预过滤
- [ ] 净化方法适合可用工具和条件
- [ ] 煮沸达到并保持了海拔调整后的滚沸时间
- [ ] 化学处理使用了正确的剂量和等待时间

- [ ] 净化了足够满足每日水合需求的水量

## 常见问题

- **跳过预过滤**：沉淀物降低化学处理效果并堵塞过滤器。浑浊水必须先预过滤
- **煮沸不充分**：锅底的几个气泡不是滚沸。等待剧烈的、冲破水面的大气泡
- **忽视海拔**：高海拔水的沸点更低。相应增加煮沸时间
- **化学剂量不足**：冷水或浑浊水需要更多化学品或更长的接触时间
- **交叉污染**：使用同一个容器装生水和净化水，或用脏手触摸饮水口
- **对最差水源只用单一方法**：对于死水或靠近牲畜的水源，使用两种方法（如过滤 + 化学，或煮沸 + 化学）
