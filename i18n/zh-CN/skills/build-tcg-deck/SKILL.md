# 构建 TCG 卡组

从原型选择到最终优化，遵循适用于宝可梦 TCG、万智牌、血与肉等主要 TCG 的结构化流程构建集换式卡牌游戏卡组。

## 适用场景

- 为特定锦标赛格式或休闲游戏构建新卡组
- 适应变化的环境调整现有卡组
- 评估新卡牌或新系列发布是否需要更改卡组
- 教授卡组构建原则
- 将卡组概念转化为锦标赛级别的列表

## 输入

- **必需**：卡牌游戏（宝可梦 TCG、万智牌、血与肉等）
- **必需**：格式（标准、扩展、近代、薪传、闪电战等）
- **必需**：目标（竞技锦标赛、休闲游戏、预算构建）
- **可选**：偏好的原型或策略（快攻、控制、连击、中速）
- **可选**：预算限制（最大花费、已有卡牌）
- **可选**：当前环境快照（顶级卡组、预期对手分布）

## 步骤

### 第 1 步：定义原型

选择卡组的战略定位。

1. 识别当前格式中的可用原型：
   - **快攻**：通过早期压力和高效攻击者快速获胜
   - **控制**：高效解决威胁，在后期通过卡牌优势获胜
   - **连击**：组装特定卡牌组合以获得强力协同或即时胜利
   - **中速**：灵活策略，根据需要在快攻和控制之间切换
   - **节奏**：通过高效的时机和干扰获得资源优势
2. 基于玩家偏好、环境定位和预算约束选择原型
3. 确定 1-2 个主要胜利条件
4. 清楚陈述原型选择和胜利条件

**预期结果：** 明确的原型和已定义的胜利条件。策略足够具体以指导选卡，又足够灵活以适应变化。

**失败处理：** 如果没有合适的原型，从最强的单卡开始，让原型从卡池中自然浮现。

### 第 2 步：构建核心

选择定义卡组策略的卡牌。

1. 确定**核心引擎**（12-20 张，取决于游戏）
2. 添加**辅助卡牌**（8-15 张）：寻找或保护核心引擎
3. 添加**互动卡牌**（8-12 张）：去除对手威胁、干扰对手策略
4. 填充**资源基础**（因游戏而异）：
   - 万智牌：地牌（60 张卡组通常 24-26 张）
   - 宝可梦：能量卡（8-12 基本 + 特殊）
   - 血与肉：投掷值分配（平衡红/黄/蓝）

**预期结果：** 达到或接近格式最低卡组大小的完整卡组列表。每张卡都有明确角色。

**失败处理：** 如果卡组列表超过格式大小，先削减最弱的辅助卡。如果核心引擎需要太多卡牌（>25），策略可能太脆弱——简化胜利条件。

### 第 3 步：分析曲线

验证卡组的资源分布是否支持其策略。

1. 绘制**法力/能量/费用曲线**并验证匹配原型
2. 检查**颜色/类型分配**：资源基础能否可靠地按曲线施放卡牌？
3. 验证**卡牌类型平衡**
4. 如果曲线不支持策略则调整

**预期结果：** 平滑的曲线，让卡组按时执行策略。

**失败处理：** 如果曲线不均匀，将昂贵的辅助卡替换为更便宜的替代品。曲线比任何单张卡都重要。

### 第 4 步：环境定位

评估卡组对预期对手的表现。

1. 识别当前环境的前 5 套卡组
2. 对每套顶级卡组评估：有利（+1）、持平（0）、不利（-1）
3. 计算对阵预期胜率
4. 如果定位不佳，考虑调整互动卡或备牌

**预期结果：** 清晰了解卡组在环境中的位置。有利和不利对局已识别并有具体原因。

**失败处理：** 如果环境数据不可用，专注于多功能性——确保卡组能与多种策略互动。

### 第 5 步：构建备牌

为格式特定的适应构建备牌/副牌组（如适用）。

1. 对每个不利对局识别 2-4 张能显著改善的卡牌
2. 对每张备牌知道：用于哪些对局、替换主牌的什么、是否改变曲线
3. 验证备牌不超过格式限制
4. 确保每张备牌至少覆盖 2 个对局

**预期结果：** 有针对性的备牌，有意义地改善最差对局而不稀释主要策略。

**失败处理：** 如果备牌无法修复最差对局，卡组可能在当前环境中定位不佳。考虑调整核心策略。

## 验证清单

- [ ] 原型和胜利条件明确定义
- [ ] 卡组符合格式合法性（禁卡表、轮换、卡牌数量）
- [ ] 每张卡都有定义的角色（核心、辅助、互动、资源）
- [ ] 法力/能量曲线支持策略速度
- [ ] 资源基础能可靠地按曲线施放卡牌
- [ ] 环境对局已评估并有具体理由
- [ ] 备牌针对最差对局并有明确替换计划
- [ ] 满足预算约束（如适用）

## 常见问题

- **胜利条件太多**：一套有 3 种获胜方式的卡组通常哪种都做不好。专注于 1-2 种
- **曲线盲区**：添加强力昂贵卡牌而不检查卡组能否按时施放
- **忽略环境**：在真空中构建。理论上最好的卡组会输给实践中最常见的卡组
- **感情用事**：保留不服务于策略的心爱卡牌。每个卡位都必须证明其价值
- **备牌事后想法**：用剩余卡牌最后构建备牌。备牌是卡组的一部分，不是附录
- **过度针对**：用狭窄的答案填满卡组而非主动策略
