# 維手工

以磨、柄養、防鏽、季存維 8 園手工：繞剪、掘刀、手叉、鏟、鋸、磨石、澆壺、耙。

## 用

- 各園會後（速清法）
- 生長季月維（磨、油）
- 季末（冬存備）
- 春前（預季備查）
- 工覺鈍、硬、或鏽

## 入

- **必**：待維園手工
- **必**：磨石（1000/3000 粒組合石或金剛板）
- **必**：輕機油或山茶油（tsubaki 油）
- **可**：鋼絲刷、細鋼絨
- **可**：亞麻油（木柄）
- **可**：220 粒砂紙（柄重修）
- **可**：替柄或五金

## 行

### 一：知八要工

此為善維園之核手工。無電動：

```
八要園手工：
┌───┬─────────────────────┬──────────────────────────────────────────┐
│ # │ 工                  │ 主用                                      │
├───┼─────────────────────┼──────────────────────────────────────────┤
│ 1 │ 繞剪（手剪）         │ 2cm 徑內活莖。園中最用之工                  │
├───┼─────────────────────┼──────────────────────────────────────────┤
│ 2 │ 掘刀（hori-hori）    │ 和刀。掘、切、除草、移植、量深               │
├───┼─────────────────────┼──────────────────────────────────────────┤
│ 3 │ 手叉                 │ 鬆土、連根起草、入改良                      │
├───┼─────────────────────┼──────────────────────────────────────────┤
│ 4 │ 鏟                   │ 植、移、舀堆肥、挖小洞                     │
├───┼─────────────────────┼──────────────────────────────────────────┤
│ 5 │ 修鋸（摺或固）       │ 剪外木切（2-10cm 徑枝）                    │
├───┼─────────────────────┼──────────────────────────────────────────┤
│ 6 │ 磨石（組合）          │ 諸刃工維。1000/3000 粒組合石多需已足        │
├───┼─────────────────────┼──────────────────────────────────────────┤
│ 7 │ 澆壺（長頸 8-10L）    │ 精澆。長頸及遠、細嘴育苗                    │
├───┼─────────────────────┼──────────────────────────────────────────┤
│ 8 │ 耙（弓或平頭）        │ 床備、平、籽床終、輕耕                      │
└───┴─────────────────────┴──────────────────────────────────────────┘

質原則：
買可負之最佳，善維，數十年。善維中檔剪勝忽上品。
```

得：熟核工集與各工主功。

敗：算限→按此序優：剪 → 鏟 → 手叉 → 掘刀。此四覆 90% 園任。

### 二：用後速清（2-3 分）

每置工時行：

```
用後法：
1. 以布或草擦諸金面土
2. 粘汁（尤剪）：以醇或 WD-40 擦
3. 乾諸金面——鋼上水=24 時內鏽
4. 返工至掛存（非地上堆）

30 秒剪清：
1. 全開剪
2. 以油布擦兩刃
3. 一滴油於樞軸
4. 開閉 3-4 次以布油
5. 閉存

此費 30 秒，加工壽年。
```

得：用後工淨乾返專存。

敗：鏽已始（鋼上橙點）→尚未晚。見步三修補。

### 三：月維——磨與油

生長季月磨諸刃工：

```
磨法：

繞剪（繞型）：
1. 可能則拆（除中栓分刃）
2. 識斜刃（切刃——僅一側磨）
3. 磨石浸水 10 分
4. 斜面平置 1000 粒石面
5. 以現斜角（典 20-25°）推刃前——5-8 劃
6. 翻石至 3000 粒——3-5 終劃
7. 以刃之平面平置石作一輕過以除絲緣（毛刺）
8. 合、油樞、試切綠枝——當淨切

   ⚠️ 勿磨繞剪之平（砧）面。
   僅斜刃磨。

掘刀：
1. 二刃皆斜——二皆磨
2. 循現斜角（常 15-20°）
3. 各側 1000 粒 5 劃，3000 粒 3 劃
4. 試：當淨切板紙

修鋸：
- 無經驗勿自磨修鋸齒
- 現代修鋸多硬脈衝硬齒
- 不效切時換刃（其為消耗）
- 僅清與油刃

手叉/鏟/耙：
- 此不需磨（為土工非切）
- 鋼絲刷除土
- 輕油塗諸金面

通油：
- 用山茶油（tsubaki）——傳日工油，食品級
- 替：輕機油或礦油
- 布塗薄層——無滴
- 重：刃面、樞、彈機
- 勿用植物油（腐而招蟲）
```

得：諸刃工銳以淨切。諸金面輕油。

敗：刃有缺或切口→需更劇磨——先以粗石（400-600 粒）始後以 1000/3000 終。深損或需職磨。

### 四：除鏽

為忽之工：

```
除鏽法：
1. 白醋浸 2-4 時（重鏽過夜）
2. 以細鋼絨或鋼絲刷擦
3. 以淨水沖並即——徹乾
4. 若有坑→輕砂 320 粒
5. 乾後即塗油
6. 木柄受濕→全乾後砂重油（見步五）

防（常）：
- 油桶法：5 加侖桶裝砂，入 1 杯礦油混。用後淨工插油砂中。砂清油護
- 掛工，勿堆。氣循阻濕積
- 存於乾處——非無暖戶外棚（溫波中凝濕）
```

得：鏽除，工效，護油塗。

敗：鏽深坑刃（見坑）→工失鋼或不持銳。考換——深坑剪刃可損植物織招病。

### 五：柄養

木柄需年注：

```
木柄法：
1. 察裂、分、鬆
   - 鬆柄：緊箍（金圈）或重坐柄頸
   - 裂柄：換——裂柄負載時斷可致傷

2. 220 粒砂紙輕砂
   - 循紋
   - 除凸紋、粗、刺
   - 勿過砂——乃平非重塑

3. 塗亞麻油（生或熟）：
   - 熟亞麻油乾速（24-48 時）有添加
   - 生亞麻油純而需 3-7 日固
   - 以布塗——薄層入紋
   - 二層，隔 24 時
   - 15 分後擦余——積油成粘

4. 用前全固（至少 48 時）

⚠️ 安全：亞麻油布可自燃。
油後布平攤戶外乾或浸水妥棄。勿團油布於桶中。
```

得：平滑油終柄，水滑阻泡。

敗：柄逾修（深裂、腐）→品牌有替柄。多剪、叉、鏟廠售替柄。

### 六：冥想點——冬工養儀

季末工維乃思修：

```
冬工養儀：
非急。取 1-2 時靜冬日。

冥想（5-10 分）：
1. 八工置淨桌
2. 坐之。此為汝年工之器
3. 憶各工此季所行：
   - 6 月剪薔薇之剪
   - 9 月分鳶尾之掘刀
   - 10 月植 200 蒜瓣之鏟
4. 認工。認手。
5. 備時慢細一工行維

全冬維序：
各 8 工：
1. 徹清（鋼絲刷、布、醇除汁）
2. 察損（鬆柄、彎齒、磨刃）
3. 可修則修（緊、重坐、換件）
4. 磨諸刃（全磨法——二粒）
5. 油諸金面（較月重——冬存）
6. 油諸木柄（亞麻油二層）
7. 掛於乾存栓——工勿相觸

畢：
諸工銳、油、掛後，園活年終。工待備春。

啟種錄瀏——工早備。
```

得：諸工峰態，冬存善，園人入休眠季。

敗：冬維全略→春前急清：鋼絲刷諸鏽，磨諸，油諸。愛工永不晚。

## 忌

1. **磨剪誤側**：僅斜刃磨。磨平面生隙致壓非切
2. **用植物油代礦/山茶**：植物油腐招蟲成粘。必用礦基油於工
3. **工存土中**：有園人插工於砂盆——佳若砂油（防桶），差若乾砂（擦與濕留）
4. **忽鬆柄**：劃中脫柄之工為險。月察柄
5. **略用後清**：「吾後行」→晨有鏽。30 秒清乃最要維慣
6. **買劣工**：不持刃之劣剪於挫與植損中費更多於持數十年之品工
