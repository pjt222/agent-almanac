# Vishnu Bhaga

存養所行——錨已驗、擾下持一、護功式於不必之變。

## 用

- 行法因蔓或早優將擾→用
- 境漂將以舊假覆已驗→用
- 多並憂壓變應穩者→用
- `shiva-bhaga` 散後——存者於重建中須護→用
- 長會將以壓失早決→用
- 改現正功之系前→用

## 入

- **必**：欲存之現工態或已驗（隱式可得）
- **可**：特穩脅（如「蔓」「壓近」）
- **可**：MEMORY.md 與案檔為接地（經 `Read`）

## 行

### 一：清存

護前辨何今功且驗。

```
Preservation Inventory:
+---------------------+---------------------------+------------------------+
| Category            | Verification Method       | Anchoring Action       |
+---------------------+---------------------------+------------------------+
| Verified Facts      | Confirmed via tool use    | Record source and      |
|                     | (file reads, test runs,   | timestamp; do not      |
|                     | API responses)            | re-derive              |
+---------------------+---------------------------+------------------------+
| Working Code        | Tests pass, behavior      | Do not refactor unless |
|                     | confirmed, user approved  | explicitly requested   |
+---------------------+---------------------------+------------------------+
| User Requirements   | Explicitly stated by      | Quote directly; do not |
|                     | the user in this session  | paraphrase or infer    |
+---------------------+---------------------------+------------------------+
| Agreed Decisions    | Decisions made and        | Reference the decision |
|                     | confirmed during this     | point; do not revisit  |
|                     | session                   | without new evidence   |
+---------------------+---------------------------+------------------------+
| Environmental State | File paths, configs,      | Verify before assuming |
|                     | tool availability         | unchanged              |
+---------------------+---------------------------+------------------------+
```

1. 每類列今驗且功之特項
2. 註驗法——汝何以知此真？
3. 無驗者非存——為假（或須 `shiva-bhaga`）

得：具體存清附證據。

敗：清稀——少驗——本身即寶信。行 `heal` 重接地乃存未驗假。

### 二：辨擾源

名脅穩態之力。

1. **蔓**：任擴超約乎？
2. **境漂**：早事為近（或誤）推覆乎？
3. **優壓**：覺欲改足用者乎？
4. **外變**：境變乎（檔改、工失）？
5. **壓險**：話近境限將失早決乎？

各源評：真脅或預險？

得：擾源命名附嚴評（活脅 vs 預險）。

敗：無顯擾源→存或不需——考 `brahma-bhaga`（建）或續行更宜。

### 三：錨穩態

施特技以護所行於辨脅。

1. **記憶錨**：境漂險之關鍵事實，再明述：
   - 「立事：[X]，由 [法] 於 [話點] 驗」
   - 持久記憶可用→書持久事實於 MEMORY.md
2. **範界護**：蔓→重述約範：
   - 「約範：[原請]。今工於此界內/外。」
3. **抗變**：優壓下之功碼：
   - 「此件功且驗。除用請外不改。」
4. **態快照**：壓險→建心檢點：
   - 摘：何已成、何餘、關鍵決何
5. **境驗**：外變→續前再察：
   - 重讀關鍵檔，勿賴前讀

得：諸辨脅有特錨應。穩態明護。

敗：錨覺過——皆等護——須先後。何最不可變？先護之。

### 四：行中持

存非被——須續工中持注。

1. 每行前察：「此脅存清中物乎？」
2. 是→覓替法達標而不擾穩態
3. 不可避→明認並更清
4. 複雜行後尤其，定期再驗存項
5. 任畢→確存項仍完

得：工態經今任完存。改僅施於需處而未擾功件。

敗：存項不慎變→立評損。改破物→還。改中性→更清。勿留舊清。

## 忌

- **以假為事護**：唯驗者堪護。未驗假裝事生偽穩
- **過存**：皆等護阻必變。存須擇——護所行、釋無功
- **被存**：假物不需活驗自穩。境漂恆；存須續注
- **抗正當變**：以存為借不施必改。用請改功件→蓋存
- **舊清**：新訊至而不更清。清須映今實，非建時態

## 參

- `shiva-bhaga` — 滅先於存；存散後者為 Vishnu 養
- `brahma-bhaga` — 建於存基；新式生於穩地
- `heal` — 子系評揭真功 vs 表穩
- `observe` — 持中觀於漂脅穩前察
- `awareness` — 境覺（Cooper 色碼）直對擾測
