# 合氣道

立中、合來力、以技解爭，非以強對強。

## 用

- 學防身術重緩、控解→用
- 引怒而不傷→用
- 安受身（ukemi）→用
- 壓下養靜中→用
- 補太極或冥（見 `tai-chi`、`meditate`）→用
- 對多敵之覺與應→用

## 入

- **必**：練所、有墊（榻、柔道墊、草；忌混凝）
- **必**：寬衣（道服佳；鬆衣可）
- **可**：練伴（技必、獨可練中、ukemi、行）
- **可**：練器（木：杖約 128cm、劍約 102cm、刀約 30cm）
- **可**：階（始、中、上；默始）
- **可**：練時（最少 20 分；薦 60-90 分）

## 行

### 一：接地立中

技皆始於中。無中則為肌爭。

1. 立半身：一足前，足距肩寬，60 度角
2. 微屈膝——重沉至「一點」（臍下五公分）
3. 全鬆肩——降離耳
4. 覺向外延而注於一點
5. 驗中：伴輕推胸
   - 上身抗→未中，鬆而沉
   - 不費而穩→中已立
6. 練重於下：想各臂之重在下；上面輕浮
7. 延氣：靜正之能透指出——非緊乃定意

得：穩鬆之姿，推由構入地非肌抗。心靜覺廣。

敗：姿剛→存緊。抖臂腿、深呼五次、重始。推易移→沉重心、注一點。立中乃月積之技，初搖常事。

### 二：精 ukemi（受身）

ukemi 乃安受技之術。aikido 第一技——將跌千次。

```
Ukemi Progression:
┌────────────────────┬──────────────────────────┬───────────────────────────┐
│ Level              │ Technique                │ Practice Method            │
├────────────────────┼──────────────────────────┼───────────────────────────┤
│ 1. Backward fall   │ Sit down, roll back,     │ From seated, then squat,  │
│ (ushiro ukemi)     │ slap mat with both arms  │ then standing. Chin to    │
│                    │ at 45 degrees            │ chest — never hit head    │
├────────────────────┼──────────────────────────┼───────────────────────────┤
│ 2. Side fall       │ Fall to the side, arm    │ From kneeling, then       │
│ (yoko ukemi)       │ slaps mat, body in arc   │ standing. Land on the     │
│                    │ — not flat on the back   │ fleshy side, not hip bone │
├────────────────────┼──────────────────────────┼───────────────────────────┤
│ 3. Forward roll    │ Roll diagonally over     │ From kneeling, then       │
│ (mae ukemi)        │ shoulder: hand-forearm-  │ standing, then moving.    │
│                    │ shoulder-opposite hip    │ The line is diagonal,     │
│                    │                          │ never straight over spine │
├────────────────────┼──────────────────────────┼───────────────────────────┤
│ 4. Breakfall       │ High fall received with  │ Only after forward roll   │
│ (tobi ukemi)       │ a slap and roll at speed │ is completely smooth.     │
│                    │                          │ Build height gradually    │
└────────────────────┴──────────────────────────┴───────────────────────────┘
```

要律：

1. 拍以散力於臂面——時於身落前
2. 各跌收頷——首勿觸地
3. 落時呼；屏息致剛、增傷
4. 前滾須圓滑——有撞或平則身未夠彎
5. 兩側皆練：右肩前、左肩前

得：常練 2-3 月後前滾兩側皆滑靜。後跌已自動（無懼）。中速可受擲不豫。

敗：前滾肩痛→角過陡（過肩頂非斜過背）。請伴或師察線。畏滾→回跪式漸建。前滾未熟勿強為破跌。

### 三：練基技

四基技應常擊景而體 aikido 核律。

```
Technique Selection by Attack:
┌─────────────────┬──────────────────┬──────────────────────────────────────┐
│ Attack          │ Technique        │ Principle                            │
├─────────────────┼──────────────────┼──────────────────────────────────────┤
│ Straight strike │ Irimi-nage       │ Enter behind the attack line, lead   │
│ (shomen-uchi)   │ (entering throw) │ attacker's head in a spiral, project │
├─────────────────┼──────────────────┼──────────────────────────────────────┤
│ Diagonal strike │ Shiho-nage       │ Redirect the attacking arm overhead  │
│ (yokomen-uchi)  │ (four-direction  │ in a spiral, control the wrist, cut  │
│                 │ throw)           │ down to throw in any direction       │
├─────────────────┼──────────────────┼──────────────────────────────────────┤
│ Wrist grab      │ Kote-gaeshi      │ Blend with the grab energy, apply    │
│ (katate-dori)   │ (wrist turn)     │ outward wrist rotation to unbalance  │
│                 │                  │ and project the attacker             │
├─────────────────┼──────────────────┼──────────────────────────────────────┤
│ Lapel/shoulder  │ Ikkyo            │ Control the elbow and wrist, pin     │
│ grab (ai-hanmi) │ (first teaching) │ the arm to the ground. Foundation    │
│                 │                  │ for all immobilizations              │
└─────────────────┴──────────────────┴──────────────────────────────────────┘
```

各技：

1. 始以慢協速
2. 先合：身離擊線（勿頭對頭格）
3. 接：觸伴臂或身以感其衡
4. 引：以圓動導敵能入技
5. 畢：擲或鎖以控——意在止爭非傷
6. 各側 10-20 次，輪角（投/取擲，受擊而跌）
7. 兩伴熟則漸增速與抗

得：技滑而圓。敵獲引非強。協速下兩伴皆安畢、無緊。

敗：技需大力→合未足，防者抗擊非引。回觸時、孤練初合。敵感拽絞→緩之、注以中引非以手拉。

### 四：養合（tai sabaki）

身行——aikido 之機。無行之技乃摔角。

1. **入身（irimi）**：直前過敵，入其達內
   - 練：伴出拳；步過之至其肩後
2. **轉身（tenkan）**：以前足旋 180 度，導敵線
   - 練：伴執腕；滑旋至同向
3. **入轉**：合入與旋——最常之行
   - 練：入過擊、旋以自後控敵
4. **回轉（kaiten）**：全身旋以生圓力為擲
   - 練：合 shiho-nage 與他旋技
5. 諸行先獨練：步旋、步旋、步旋遍墊
6. 後合伴：注時於擊——其決時行，非前非後

得：行流、中、合擊時。防者永不正對擊力——恆斜離線。

敗：時誤（過早或晚）→以極慢明擊練。防者於擊決時動——此乃「合氣」之時。轉鈍→獨練 tenkan：各課百旋以建滑自動。

### 五：練亂取（多敵）

亂取養壓下之覺與決。律真受驗於此。

1. 始以二慢敵——輪番（非同時）
2. 核：永勿停。靜即被圍
3. 用各敵為盾擋他：引一入他之路
4. 行至空地——勿退角或牆
5. 覺延諸向；勿執一敵（見 `mindfulness`）
6. 用簡可靠之技——複雜壓下敗
7. 進至三、四敵，速漸增
8. 亂取非為勝諸敵——乃存中與行之自由

得：多向被近時能靜能行。技流暢無凍無管視。

敗：起恐→回二慢。亂取慮常事，露次而減。技壓下崩→簡之：唯入身、轉身，棄擲。亂取中行與位重於施技。

### 六：合器覺

器練深距、時、線之解——益空手技。

1. **杖**：練 31 數杖型以線與延
   - 杖教存距、控空
   - 伴練：奪杖——以空手解杖擊
2. **木劍**：練素振以精度與中線覺
   - 劍教決：每砍須循精線
   - 伴練：對劍以時與距
3. **木刀**：練奪刀
   - 刀教近危之敬
   - 永引器手——勿伸刃
4. 器律轉空手：
   - 間合（正距）：太近招器；太遠廢應
   - 正面（中線）：諸擊防皆關於垂中線
   - 殘心（續覺）：技畢後仍警

得：器練明 tai sabaki、時、距之要。空手技進，身行更精。

敗：器拙或斷於空手→注素振一月再合伴。伴練成競或危→即緩——木器於速可真傷。

### 七：施律於墊外

aikido 之值越道場入日交與爭解。

1. 對峙時心施入轉：向其憂行（入），轉話至共地（轉）
2. 物空中存位、出口、他身語之覺（見 `mindfulness`）
3. 練「受」評或怒不抗——認其能、後引
4. 養壓前立中之習：覺降一點、鬆肩、延靜意
5. 常練表：
   - 獨（日 15-30 分）：立中、ukemi 滾、tai sabaki、器素振
   - 伴（週 2-3 次，60-90 分）：技、亂取、器伴練
6. 補太極為內能（見 `tai-chi`）
7. 補冥為心靜（見 `meditate`）

得：合、引、立中律成日爭之自然應。身練存且深武技。

敗：日練斷→注最小：5 分立中、20 前滾。常勝久。武似斷於日→思各技為喻：irimi 直面題；tenkan 改見；ukemi 復於敗。

## 驗

- [ ] 中姿受中推無肌抗
- [ ] 前滾兩側皆滑靜
- [ ] 至少四基技可協與伴行
- [ ] tai sabaki（irimi、tenkan、入轉）獨與伴皆練
- [ ] 二慢敵亂取可持 60 秒不凍
- [ ] 至少一器（杖、木劍、木刀）獨型練過
- [ ] 練含獨與伴常分

## 忌

- **以肌代合**：技需力→時或角誤。鬆、再入、令敵能為功。aikido 之力出引非生
- **忽 ukemi**：避跌者避學。ukemi 即 aikido——安受力之能。每課練、尤始
- **畏決**：半 irimi 致最劣位——太近不避太遠不控。決入則全行過擊
- **執一敵**：亂取與生中管視危。持軟廣覺。周視察動先於焦視
- **唯協練**：始須協，中當漸加抗。唯協能行之技未全

## 參

- `tai-chi` — 補內武術；共讓克強之律，重獨修
- `mindfulness` — 防情境覺供武備與避爭之知基
- `meditate` — 坐冥養 aikido 壓下所需之中、平心
- `heal` — aikido 解體機益急救與身工
- `redirect` — AI 自施變；映 aikido 合與引於處衝需與工敗
