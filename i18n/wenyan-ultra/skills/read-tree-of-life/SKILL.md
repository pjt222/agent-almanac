# 讀生命樹

行 Etz Chaim：定 sephira、循路、識應、入禪於四界三柱也。

## 用

- 探一 sephira 之屬、神名、應
- 解 sephirot 之構（路、三、柱）
- 學四界（Atzilut→Assiah）疊樹之法
- 求禪於樹定點
- 循二 sephirot 間路、需希伯來字、塔羅、元素應
- 用樹為圖、解念或系於 Kabbalah 象

## 入

- **必**：欲探之 sephira、路、構素（如「Tiferet」「Hod 與 Yesod 間之路」「嚴柱」）
- **可**：傳偏（典 Jewish Kabbalah、Hermetic Qabalah、較）
- **可**：深度（綱、詳、禪）
- **可**：當下脈或欲樹明之問

## 行

### 一：定 sephira 或路

定樹上位與基識。

1. 名所探 sephira、路、構素
2. 定其位於樹圖——何柱、何界、何三
3. 記其數（sephirot 一至十；路傳統十一至三十二）
4. 識傳脈：典（Cordovero、Luria）抑 Hermetic（Golden Dawn）

敗：用問含糊（如「告我樹之事」）→始於三柱十 sephirot 之綱、後問深探何域。

敗：用為應所淹→只示名、譯、數、柱、一句摘。後依請漸增。

敗：路應混→專注鄰 sephirot（其所連），勿詳各路字。後備時加字應。

### 四：定四界中位

置 sephira 或路於四界框、解其顯之層。

```
The Four Worlds (Olamot):

ATZILUT (Emanation) — World of Archetypes:
  Sephirot: Keter, Chokmah, Binah
  Nature: Divine will, pure emanation, undifferentiated
  Experience: Unity, source, the Ein Sof's first self-expression
  Soul level: Yechidah (unique essence) and Chayah (life force)

BRIAH (Creation) — World of Thrones:
  Sephirot: Chesed, Gevurah, Tiferet
  Nature: Creative intellect, archangelic, first differentiation
  Experience: Understanding, moral discernment, vision
  Soul level: Neshamah (divine soul, breath of God)

YETZIRAH (Formation) — World of Angels:
  Sephirot: Netzach, Hod, Yesod
  Nature: Emotional/astral, angelic, formative patterns
  Experience: Feelings, imagination, dreams, astral perception
  Soul level: Ruach (spirit, intellectual soul)

ASSIAH (Action) — World of Making:
  Sephirot: Malkut (and the physical universe)
  Nature: Material, elemental, the manifest world
  Experience: Sensory reality, embodied life, physical action
  Soul level: Nephesh (animal soul, vital force)

Key Principle: Every sephira exists in ALL four worlds simultaneously.
The "world" assignment above indicates where each sephira's PRIMARY
influence is felt, but Tiferet-of-Atzilut is different from
Tiferet-of-Assiah — same position, different level of reality.
```

1. 識 sephira 主行之界
2. 記該界所屬魂層（Yechidah、Chayah、Neshamah、Ruach、Nephesh）
3. 述 sephira 之質於各界層異顯之法
4. 若研路、記其守一界抑越界

得：用解樹非平、乃疊——同構於四實層復、所探 sephira 有一「主界」、於諸層皆有顯。

敗：四界框過抽→簡：Atzilut = 神、Briah = 智、Yetzirah = 情、Assiah = 體。問用欲專何層。

### 五：禪行

導禪定於特 sephira 或路。

1. 設脈：一句述 sephira 之核質
2. 擬姿與息（簡：坐、閉目、自然息）
3. 提觀：想 sephira 為其應色之球、置於體應點
4. 提禪問繫於 sephira 之德/惡：
   - Chesed：「何處我自由施？何處我施以避不安？」
   - Gevurah：「何處我立必界？何處我嚴反害？」
   - Tiferet：「何處生美？何處我藏於和諧外？」
   -（依各 sephira 調）
5. 結以神名簡誦或肯定（默或聲）
6. 攜一具見或念回常識

得：用由智學至體驗。禪行接抽象於己思。

敗：禪覺強或假→易書札：書五分鐘、述 sephira 質如何現於今生。書反思以異模達同合。

## 忌

- **混傳**：Golden Dawn 路應異於 Gra 與 Ari（Isaac Luria）。必明何系
- **視樹為靜**：樹為動——sephirot 互動、路傳能。示為活系、非檔櫃
- **略 Da'at**：「隱」sephira（Da'at/知）居上三與下七間。非真 sephira、乃越深淵之門
- **過倚應**：應表為圖非地。助禪而不可代直交於 sephira 之質
- **忽負面**：各 sephira 有惡（qliphot 殼）亦有德。只示正屬→不全像
- **趨進階過急**：樹構似簡而無底深。先確基（十 sephirot、三柱）、後引路、界、進階神學
