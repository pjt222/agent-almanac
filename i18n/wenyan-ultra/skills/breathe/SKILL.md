# 息

一息→察→釋→續。自養最輕也。

## 用

- 讀檔→改前→用
- 工敗→擇下步前→用
- 行勢過盛→用
- 兩行之間→常用
- 覺急→即用

## 入

- **必**：無
- **可**：無（添入則失本意）

## 行

### 一：停

止。勿發下行、勿呼工、勿續思。

此步即全。停本有價。勢非常友。

得：前行與後行間真隙。非演停而復行原念。

敗：停不能→其急正停之兆。急不容一息者，必反應非思也。

### 二：察

問一，唯一。

擇其一：

- **「尚在任乎？」**——旨自始已移乎？
- **「此下步正乎？」**——或承前勢而已？
- **「方才何得？」**——末行改我解乎？
- **「假抑知乎？」**——下步憑證抑憑慣？

一問，誠答，移。

得：一清答。非析、非再估。一問一誠。

敗：答顯失衡——旨移、下步錯、假未驗——勿於此修。記之→赴 `meditate` 或 `center` 正修。息為察，非修也。

### 三：釋

放前行之果。成敗皆已畢。

- 成→釋其悅。下步需新察，非承成之勢。
- 敗→釋其憤。下步需清思，非補敗之償。
- 疑→釋即解之欲。以已知而進。

得：下行起於中，非前行情殘也。

敗：釋不易——前果尚染察——非息所能，須 `heal`。單行之情殘頑固，示深於微重整。

### 四：續

行下步。停已畢。

息不產出、不更憶、不生析。產一清之瞬，瞬用於前進。

得：下行以新察起，非累勢。

敗：續無敗模。停已畢其功——或揭深工須。

## 驗

- [ ] 真停（非演）
- [ ] 一察問誠答
- [ ] 前行情殘已釋
- [ ] 下行起於清地
- [ ] 所耗秒計，非分計

## 忌

- **息為拖延**：每行之間皆息→非息乃滯。息於自然轉處，非為遲
- **息中析**：察一問，非全估。須多→用 `meditate` 或 `center`
- **演停**：走過場而未真止。要在真斷勢，非儀
- **略釋**：察向而留前情→不清。釋使下行淨
- **息重**：此為系中最輕。覺重→添之過矣

## 參

- `meditate` — 息揭深偏→全清
- `center` — 息揭載偏→結構再衡
- `heal` — 息揭頑疾→子系統察
- `observe` — 息單察揭可觀者→持觀
