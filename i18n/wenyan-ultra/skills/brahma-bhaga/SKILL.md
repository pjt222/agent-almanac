# Brahma 分

自虛或模自生造—新模、新法、新解自無而出之結構出現。

## 用

- `shiva-bhaga` 解已清陳模立空後
- 遇真新題無顯模或先例
- 用者請需造而非取或適
- 多路存而未擇—造之行即擇
- 白盤：新檔、新案、新構、新法
- 漸改已至限而需新設

## 入

- **必**：造之標或當填之虛（自會脈得）
- **可**：束造之限（用者求、技限）
- **可**：種—片、啟、部分念以鋪造
- **可**：所解之物（`shiva-bhaga` 出）—知何敗導當造何

## 行

造前→解可造之空。

1. 誠圖每維—尤限，多隱
2. 記自由度：真造生於此
3. 識種而未委之—鋪而不令

**得：** 造空之清圖：限縛、種鋪、自由度開。

**敗：** 空覺全限（無自由度）→重察—似定之限多實為偏。必則問用者。

### 二：生—發散探

出多可，未評。

1. 至少生三別法以填造空
2. 諸法當真異—非題上變
3. 每法→捕：
   - 核念一句
   - 如何合限
   - 別於他之因
   - 所捨或所換
4. 含至少一法覺非常或險
5. 未評—生與評乃別階

**得：** 三或多真異法，各有清身與得失形。

**敗：** 諸法覺似→生過狹。返步一尋未探之自由度。或倒一限：「若反顯法乎？」

### 三：評—聚擇

較生法與造空。

1. 每法→評：
   - **限合**：合諸不可商求乎？
   - **雅**：最簡而行乎？
   - **韌**：可過未擾乎？
   - **整**：合周脈乎？
   - **新**：攜真新，抑但重排舊？
2. 除違硬限之法
3. 餘法中→依用者隱值擇（簡？周？造？）
4. 若兩法等強→呈予用者附清得失

**得：** 一擇法（或為用者清擇）附述理。

**敗：** 無法合諸限→限或矛。呈矛於用者非迫造於根讓。

### 四：出—入形

行擇法，予具體形。

1. 自骨始：體核念之最簡構
2. 自核而外建，需處加詳
3. 每步→察：「此加服核或稀之？」
4. 抑過飾—造完於不可減
5. 名所造：清述 ID 捕其精

**得：** 體擇法之具造—碼、謀、構、設—存於先虛。

**敗：** 出偏擇法→停而重讀步三擇。出時偏常示未全委於擇。或重委或重擇。

### 五：育—護嫩造

新造脆。護之於初。

1. 較造與限—行如所意乎？
2. 識最弱點—何處最易破？
3. 強最弱而不過工
4. 若造當存→交 `vishnu-bhaga` 續保
5. 錄造擇：擇何、棄何、由何

**得：** 造已試、錄、備續用。

**敗：** 造初試敗→評敗於造或試。若造本缺→返步二，以敗為新反模種。

## 驗

- [ ] 造空先察而後生念
- [ ] 至少三真異法生
- [ ] 擇基顯準，非默直覺
- [ ] 造自核而外建
- [ ] 造與限較試
- [ ] 造擇為未來錄

## 忌

- **未清先造**：未先解而造→新模染於舊。若空雜→先行 `shiva-bhaga`
- **單法生**：生一法而評之—非造，乃行首念。真造需散擇
- **為新而新**：造非常而簡標即足。新為工，非標
- **完美出**：久飾而不發可行之造。完不完之造勝未完完之造
- **未護造**：出新即進，不試不錄→造危

## 參

- `shiva-bhaga` — 毀立 Brahma 所填之虛；解先於造
- `vishnu-bhaga` — 保續 Brahma 所造；自造至維之交
- `intrinsic` — 造得於自動；造盛於流
- `learn` — 造需未持之知→學先於生
- `adapt-architecture` — 自現系造新構模之形對應
