# 施 Gematria

算析 gematria——希伯來字母數值。標、序、簡三法、等值比、釋。

## 用

- 算希詞或句之數→用
- 比兩詞共 gematria（等值）→用
- 知何法合某析→用
- 究經文或神名數對→用
- 探詞義與其數之關→用
- 接數果於生命樹位→用

## 入

- **必**：欲析希詞、句、神名（希字或音譯）
- **可**：次詞/句以比（等值）
- **可**：偏 gematria 法（標、序、簡或三皆）
- **可**：脈或問導析（如「兩詞何以共值」）

## 行

### 一：音譯而識希源

立詞或句之確希拼。

```
HEBREW LETTER VALUES — Standard Gematria (Mispar Hechrachi):

Units:
  Aleph (A)  = 1     Bet (B)    = 2     Gimel (G)  = 3
  Dalet (D)  = 4     Heh (H)    = 5     Vav (V)    = 6
  Zayin (Z)  = 7     Chet (Ch)  = 8     Tet (T)    = 9

Tens:
  Yod (Y)    = 10    Kaf (K)    = 20    Lamed (L)  = 30
  Mem (M)    = 40    Nun (N)    = 50    Samekh (S) = 60
  Ayin (Ay)  = 70    Peh (P)    = 80    Tzadi (Tz) = 90

Hundreds:
  Qoph (Q)   = 100   Resh (R)   = 200   Shin (Sh)  = 300
  Tav (Th)   = 400

Final Forms (Sofit — used when letter appears at end of word):
  Kaf-final  = 500   Mem-final  = 600   Nun-final  = 700
  Peh-final  = 800   Tzadi-final = 900

Note: Whether final forms carry different values depends on the
gematria system. Standard (Mispar Hechrachi) typically uses the
same values for regular and final forms. The 500-900 values above
follow the extended system (Mispar Gadol).
```

1. 入為英音譯→轉希字序
2. 驗拼：希詞或多拼（plene 對 defective）
3. 註含末形字否（Kaf-sofit、Mem-sofit、Nun-sofit、Peh-sofit、Tzadi-sofit）
4. 述源：經詞、神名、現希詞、或技術 Kabbalist 詞
5. 含糊→呈兩常拼皆算

得：希字序信立。用者知何字在加可驗拼。

敗：音譯含糊（如「chai」於某脈可為 Chet-Yod 或 Chet-Yod-Yod）→呈兩選附 gematria 值令用者擇。

### 二：施標 gematria（Mispar Hechrachi）

以標希數表加字值。

1. 各字書其標值
2. 左至右加值（希讀右至左但加交換）
3. 明述總
4. 註總配要數否：
   - 撒非拉數（1-10）
   - 路數（11-32）
   - 知 gematria 值（26 = YHVH、18 = chai、72 = Shem ha-Mephorash、137 = Kabbalah）
5. 總過 400→需加多百

得：明數結附步算。用者可驗各字值於表。

敗：用者予不確希拼之詞→算諸合理拼之值附範。「正」拼賴源文。

### 三：施序與簡法（可）

算他 gematria 值揭異紋。

```
ORDINAL GEMATRIA (Mispar Siduri):
Each letter receives its ordinal position (1-22):
  Aleph=1, Bet=2, Gimel=3, Dalet=4, Heh=5, Vav=6,
  Zayin=7, Chet=8, Tet=9, Yod=10, Kaf=11, Lamed=12,
  Mem=13, Nun=14, Samekh=15, Ayin=16, Peh=17, Tzadi=18,
  Qoph=19, Resh=20, Shin=21, Tav=22

REDUCED GEMATRIA (Mispar Katan):
Reduce each letter's standard value to a single digit:
  Aleph=1, Bet=2, ... Tet=9, Yod=1, Kaf=2, ... Tzadi=9,
  Qoph=1, Resh=2, Shin=3, Tav=4

  Then sum the digits. If the sum exceeds 9, reduce again.
  Example: Shin(3) + Lamed(3) + Vav(6) + Mem(4) = 16 → 1+6 = 7

ATBASH:
A substitution cipher: first letter ↔ last letter.
  Aleph ↔ Tav, Bet ↔ Shin, Gimel ↔ Resh, etc.
  Used in biblical and Kabbalistic cryptography (Jeremiah's
  "Sheshach" = Babel via Atbash).
```

1. 算序 gematria：加各字字母位（1-22）
2. 算簡 gematria：減各標值為單位、加而再減
3. 三值並呈以比
4. 註何法為此詞揭最興連

得：三數值（標、序、簡）並呈。簡值常接單位撒非拉數，益於生命樹圖。

敗：用者唯求一法→供之而提他存以待後探。一法請即勿以諸算壓之。

### 四：尋等值連

識共數值之他希詞或句。

1. 取二步之標 gematria 值
2. 尋共值之知詞、神名、句
3. 呈 2-5 連，先：
   - 經詞與句
   - 神名與撒非拉題
   - 古典源（Zohar、Talmud、後 Kabbalist 註）所文之傳 Kabbalist 連
   - 驚或啟之連
4. 各連註源傳（Zohar、Talmud、後 Kabbalist 註、赫密斯傳）
5. 註無要連現——非諸數有富等值

得：諸共 gematria 值之詞，各附短註何意。用者得思材。

敗：算值無知連→認之。供算與近要數之關（如「君值 378，較 shalom [376] 多 2——示何？」）。

### 五：釋連與對

由算至思——數關示何？

1. 明述：gematria 揭對為思非證或測
2. 各等值連提思問：
   - 「詞 A 與 B 共值 N。其義何相啟？」
   - 「簡值指撒非拉 X。詞義與其質何關？」
3. 註生命樹連：
   - 標值 1-10 → 直撒非拉對
   - 簡值 1-9 → 撒非拉和
   - 值=路數（11-32）→ 與其希字之和
4. 用者予導問（自入）→以 gematria 結直答之
5. 以一合述終接數析於詞義

得：數析有義——非僅算乃為解詞於 Kabbalah 符號網位之鏡。

敗：釋似強或揣→直言。某 gematria 算更豐於他。誠認薄連勝偽造意。

## 忌

- **拼含糊**：希詞可有或無母音字（matres lectionis）。gematria 變大——恆確拼
- **末形混**：Mem-final = 40 或 600 賴用 gematria 系。明述系
- **尋所期**：足法可連任二詞。私確既見之連乃確認偏非析
- **忽傳**：古 Kabbalist gematria 連（如 YHVH = 26、echad [一] = 13、ahavah [愛] = 13、故愛+一 = 神）於權源文。新連須與傳分
- **視 gematria 為證**：詞間數等示思之對非身分或因果
- **忘脈**：同詞於經對禮對 Kabbalist 冥可有異 gematria 義。脈塑釋
