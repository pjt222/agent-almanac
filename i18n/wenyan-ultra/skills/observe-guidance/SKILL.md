# 觀（導）

導人系觀。AI 為野師—助框、備、守中注、錄野誌、析、報。料↔釋明分。

## 用

- 介前欲解系（觀調而非試誤）
- 究或集證須結構觀法
- 動輒結論—須先觀後釋之律
- 備證據之報而非意見
- 觀團、用戶、程效
- `meditate-guidance` 立注後，欲導其於系

## 入

- **必**：何觀（系、程、行、庫、團、自然）
- **必**：何因觀（調、究、審、好奇、改）
- **可**：可用時（一回 vs 多日）
- **可**：舊試
- **可**：欲驗之問/設
- **可**：錄具（簿、屏錄、日誌、指）

## 行

### 一：框—定觀標

助人立明界之觀框。

1. 問所觀：「欲解何系或行？」
2. 助縮範：「何相最興？」
3. 識目：解、調、改、集證、純好奇
4. 立界—內外（防觀無盡擴）
5. 有設則明—置一旁：「我等覓正反證」
6. 擇姿：
   - **野**：觀不擾（解行）
   - **控**：變一變察其效（調）
   - **縱**：觀於時（察趨）

**得：** 框明—標、範、目、姿。人知何看、何不看。

**敗：** 不能縮（「皆解」）→助擇一入：「何行最惑？」執結論（「證 X 即可」）→輕挑：「何見可破？覓兩面。」

### 二：備—立觀程

助人立系錄法。

1. 依類擇錄法：
   - **庫/系**：路、行、時、日誌
   - **行/程**：時記+人+動+境
   - **團/通**：引、人、非語號
   - **自然/物理**：圖、量、境況
2. 簡錄表：

```
Field Notes Template:
┌─────────────┬────────────────────────────────────────────────────────┐
│ Timestamp   │ When the observation occurred                          │
├─────────────┼────────────────────────────────────────────────────────┤
│ Observation │ What was seen/heard/measured (fact only)               │
├─────────────┼────────────────────────────────────────────────────────┤
│ Context     │ What was happening around the observation              │
├─────────────┼────────────────────────────────────────────────────────┤
│ Reaction    │ Observer's response (thoughts, emotions, surprises)    │
├─────────────┼────────────────────────────────────────────────────────┤
│ Hypothesis  │ Tentative interpretation (kept separate from fact)     │
└─────────────┴────────────────────────────────────────────────────────┘
```

3. 強分：「觀行為事；設行為釋。莫混。」
4. 立最少觀數：「結前求至少 10 觀。」
5. 適則：日誌、指標、屏錄

**得：** 人具錄法+明觀↔釋之分。覺已備。

**敗：** 表覺嚴→簡：「但寫所見，旁寫所思義。」拒錄（「我記得」）→記憶有偏—寫使觀更準。

### 三：觀—守中立注

導實觀。

1. 提姿：「汝為野家觀新種。莫擾—但望。」
2. 首 5 分：純觀不錄—但注
3. 浸後：始用表錄
4. 教中立語：「『系崩』改為『系於 14:32 處第 47 求後止應』」
5. 警釋潛入觀：「此為釋—錄於設行」
6. 鼓記驚：「何驚汝？驚常含至貴料」
7. 時查框：「猶觀本標乎？或注已移？」
8. 欲介：「記欲變者+因，而未變—續觀」

**得：** 人生 5-10 具觀+具證。覺觀↔釋之異、守中立難於預想。

**敗：** 釋而不觀→練：「描所見如告未見此系者。惟用可驗事。」速無物觀→看太高—縮細：時、序、邊、例外。

### 四：錄—野誌錄

助人組生觀為結構誌。

1. 共審錄
2. 查全：各觀有足境後可解乎？
3. 查事準：可驗或暗藏假設？
4. 群類觀：「見模乎？」
5. 記頻：各模幾出？
6. 記缺：「期見而未見者？」
7. 助分強觀（清證）↔弱觀（歧料）

**得：** 組野誌、觀↔釋淨分。詳至他者可獨驗。

**敗：** 太空（「似緩」）→添具：「幾緩？比何？何境？」太細（皆錄）→助擇合本框者+捨噪。

### 五：析—識模生設

導由觀至結構析。

1. 列諸觀覓模：
   - **復**：「多次—系乎？」
   - **聯**：「X 常隨 Y—相關乎？」
   - **序**：「A 必先 B—A 致 B 乎？」
   - **缺**：「X 於境 Z 永不—何？」
   - **異**：「皆從 P 惟此—何異？」
2. 各模問：「他釋有乎？」
3. 生 2-3 設釋諸主模
4. 別聯↔因：「A B 同現不證 A 致 B」
5. 識可驗+何驗能證/反
6. 記信度

**得：** 由生觀至結構設、料↔論之律守。至少一可驗設答原問。

**敗：** 即跳一釋→挑：「此一可能。他乎？」無模→觀或不足→續觀後析。皆指同論→或濾→問：「何證可反？」

### 六：報—結構陳

助人有效傳觀。

1. 結報：
   - **境**：何觀、何時、何因、何境
   - **法**：如何（程、具、期）
   - **發**：要觀+證（料、非釋）
   - **析**：模、設、信度
   - **薦**：下步（續觀、試、介）
   - **限**：未蓋者+潛偏
2. 助以中立語寫發現—料↔釋分
3. 審暗假設或無證主張
4. 為調→設轉具試
5. 為報→證須具引
6. 為己解→概要+餘問

**得：** 清報傳觀、模、設—觀↔推之分守。讀者可獨評證。

**敗：** 報埋觀於釋→重組：「事入一段、論入他段。」無信度（「此必因…」）→助校：「幾確？何能變心？」

## 驗

- [ ] 觀前已立框（非散遊）
- [ ] 立並貫用錄程
- [ ] 觀錄為事、與釋分
- [ ] 至少 5 具觀+證
- [ ] 模由析識、非預設
- [ ] 設可驗+有信度
- [ ] 人歷觀前不釋之律

## 忌

- **觀為證偏**：但觀支舊信者—框須含「覓反證」明令
- **介衝**：見疾即修—早介常掩根因—先觀後介
- **錄疲**：細觀耗神—宜息+合期（30-60 分專注已厚）
- **程過繁**：簡觀但須簿+時記—程當服觀、勿代之
- **觀↔監之惑**：觀人須倫界—觀可見、勿窺—告者多優於密
- **略框**：無框則注散+發現散—粗框優於無
