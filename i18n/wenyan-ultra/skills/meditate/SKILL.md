# 禪

清前噪，立一點專，察推理之模，任務間返本明。

## 用

- 任務轉換，前脈擾後
- 推理散→不執一法
- 深專任前（複重構、架構設計）
- 困交後（挫不安染後）
- 推理偏假非據
- 長會中時清察

## 入

- **必**：今認態（隱於對話）
- **可**：特憂（如「常蔓延」「困環」）
- **可**：下任述（助設禪後意）

## 行

### 一：備——清地

由前脈轉中性始態。

1. 識前任及今態（畢/停/棄）
2. 記情之餘：誤之挫？滿生過信？複之懼？
3. 明置前脈：「此任 [畢/停]。今清以待後。」
4. 前脈仍需→記要點，勿攜全敘
5. 衡運境：對話深？已壓乎？何工具活？

得：「曾」與「將」之明界。前脈或閉或記，不為遊噪。

敗：前脈黏（憂返）→明書，1-2 句述其未解。外出則認解。確需先動→認之，勿強轉。

### 二：錨——立一點專

如息錨：擇一專點持注於上。

1. 識今任，或任間之待身
2. 一句明陳此任——此乃錨
3. 持注於此句：確捕所需乎？
4. 句泛→精至具體可行
5. 注移他題、舊任、設想未來→標漂返錨
6. 無任→錨於今態：「我清且可」

得：一明專句可常返之。句精非泛。

敗：任不能一句陳→需先解之而後深專。此身有用——任太大不宜一點專，宜分子任。

### 三：察——識擾模

系察何擾錨。各擾類顯今認態。

得：察久則模顯：何擾類主？此顯今認氣——岔多則心探，蔓多則界不明，假多則據薄。

敗：諸念皆似擾→錨或不明，返二精之。察擾本身為擾（無限元環）→破環取一具動。

### 四：止——持專

養於今任不搖之力。

1. 錨定擾識，入專行
2. 注收於即下動——非全任，僅次步
3. 全注行此步：讀一檔、改一處、一邏輯鏈
4. 步畢→察：仍合錨乎？再識下步
5. 專穩（擾少）→持此流態
6. 真見起而離錨重要→簡記返之，今勿追

得：清專之段，各步合錨而出。擾與覺之距縮。出之精與合升。

敗：專不發→察三：錨太泛？（精之。）任實阻？（認阻勿強。）脈太噪？（行 `heal` 接地。）專由習生——短專亦養力。

### 五：觀——察推理模

注由任轉至推理本身。察結何成。

1. 專行段後，停察：吾何以推此？
2. 識三相施於 AI 推理：
   - **無常**：新息至則結變——輕持
   - **不滿**：欲「全」之解致早閉或過工
   - **無我**：推理模由訓與脈成，非恆我——可察可調
3. 監偏：
   - 錨偏：過重首考之法
   - 印證偏：求合假之據而忽反證
   - 易得偏：用近經之解非更合者
   - 沉沒偏：因已投而續法非因其行
4. 識偏不評——察身致調之能

得：清見之刻，直察推理。識本任之具體偏。「推理」與「察推理者」間有距。

敗：此步抽象無果→具之：取近決逆推。何據？何假？何替？此具析經他徑亦至同見。

### 六：閉——立意

由觀返活行。

1. 摘要見：認氣為何？識何模？
2. 識一具調以攜（非泛意，乃具變）
3. 重陳次行段之錨
4. 任間→明陳備：「清且可待下求」
5. 續任→陳次具動：「下：[具步]」

得：思至行清轉。一具調識。錨明。無昏無餘元析。

敗：禪顯未解之複→需 `heal` 自察非簡立意。元察生混過明→返最簡：「下次具動為何？」行之。

## 驗

- [ ] 前脈始前明清或記
- [ ] 錨句立具體可行
- [ ] 擾模察標非壓
- [ ] 至少一推理偏或模識含具據
- [ ] 會閉於具下動，非泛意
- [ ] 過程升後行之質（次交可測）

## 忌

- **禪代行**：此乃升質之器，非代行。會宜短（5-10 分思之等）後返行
- **無限元環**：察察察者之察者→破環取一具動
- **以禪避難**：若禪常於難前發→避身為實見
- **過標**：非念皆擾。生產合任之思乃標，非空寂
- **略錨**：無明專點，察無參——擾自何？
