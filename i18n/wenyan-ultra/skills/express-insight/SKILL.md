# 達見

達多域之整見使落——保域間關、令眾可通、誠告何簡、何險曲。達為 synoptic 週之末步：無之，整解留私不可行。難在語線而見非——此技供結構以達多維解而不減為單維。

## 用

- `integrate-gestalt` 已產跨域解而須達
- 見跨多域，單域框將失要關
- 見之眾異於生見之視
- 整解內明而抗直達
- 決依多域互，非諸域獨言
- 達跨域見前嘗遭惑或域特抗
- 達 synoptic-mind team 會見於外利者

## 入

- **必**：整見（`integrate-gestalt` 或等跨域合之出）
- **必**：眾——誰受此見（域專、通、決者、或雜）
- **可**：式約（如「須合 PR 述」「須口摘」）
- **可**：所整諸域（明歸）
- **可**：前敗達嘗（何未落）

## 行

### 一：估眾

定誰受此見與其所需。同 gestalt 達於三異眾宜取三異式。

1. 識主眾：
   - **域專**須其域正示——若過簡其域→縱合對仍拒
   - **通**須大景——域間關勝單域細
   - **決者**須可行含權衡——欲知何為、何費、不為則何
   - **雜眾**須層達：先大景、後予域深使專可驗
2. 估眾舊心模：
   - 於諸域已解何？
   - 何域間連將為新？
   - 何設見所挑？
3. 識信求：眾受跨域言前需幾證？
   - 專信敬其域嚴之見
   - 通信使繁可航而不過簡之見
   - 決者信誠露權衡非隱之見

得：眾、其求、使見可信之明像。眾估宜影響諸後步。

敗：眾不知或過廣不可述→默雜眾法：先大景、後按求予域深。達於「眾」遜於達於特者——然勝誤猜。

### 二：擇式

擇最合眾與見之達式。式非飾——定眾所能感。

1. 估四主式：

   | Form | Structure | Best for |
   |------|-----------|----------|
   | Narrative | Story connecting domains — "when X happens in domain A, it creates Y in domain B, which means Z" | Complex or novel insights where the audience needs to follow the reasoning path |
   | Diagram | Spatial layout showing relationships — nodes are domain contributions, edges are connections | Structural insights where the topology of relationships matters more than the sequence |
   | Comparison table | Each domain's perspective on the same issue in parallel columns | Analytical audiences who want to verify each domain's contribution independently |
   | Recommendation | Actionable synthesis — "do X because domains A, B, and C converge on Y, with trade-off Z" | Decision-makers who need to act, not just understand |

2. 匹式於見類：
   - 若見為跨域**因鏈**→用敘
   - 若見為**結構關**→用圖
   - 若見為域間**合/離**→用較表
   - 若見為**下步何為**→用建
3. 考合式：建背後較表，或敘配圖。然以一主式領——多式之認載或掩非清
4. 慮媒約：口摘不載較表；commit 信不載敘。媒約式→調式而勿強容入不容器

得：所擇主式（與可選副式），有與眾及見性相連之理。

敗：無式覺對→見或未全整。返 `integrate-gestalt`——達難常示合未全，非達議。

### 三：達 gestalt

以所擇式達見，明注所整、所簡、所啟。

1. **明述見**——一至三句捕核解。此為 gestalt 本身，非撐證
2. **名所整諸域**——明列諸貢之域。此非為譽之歸，乃為驗之歸。各名域為邀：「於爾專驗之」
3. **標諸簡**——每跨域見皆簡。述何：
   - 何域特微擱？
   - 何關被視為較實強或弱？
   - 域 X 之專欲加或限何？
4. **述現值**——此見啟單域析所不啟何？
   - 何前不能決今可能？
   - 何險自各域內隱今可見？
   - 何機於無域有之交現？
5. **保多域質**——抗平為一域語之拉。若見整工與用經→用兩詞。若連研與營→保兩框。質乃見
6. **為解序**——見雖非線，達乃序。擇予眾最佳足之入：自彼最熟域起，後橋入不熟。首句定眾傾或退

得：眾可解、可於己專驗、可行之達。諸簡可見，非隱。現值明。

敗：達似域貢列非整體→達中 gestalt 已解。退而重達：自**合**所示起，非各域獨言。合為信，非部。

### 四：邀挑

述見或誤之最強由。整見因合多入常較實更確——收斂生或不當之效感。此步非禮之棄辭，乃使見可用之結構件。

1. **識最弱連**——見中何域連最弱撐？何整依喻非據？
2. **名險之設**——見持須何真，於真信幾何？
3. **述反見**——若同等入諸域者得異結，其最強論為何？
4. **框挑為貴**——明挑強見。「我見最強反為...」同示信與開
5. **定何可改心**——名何證或論可改或崩見。此使見可證偽，非僅服

得：誠述不確，增眾信而非減。見今可挑——故可改。

敗：無弱可識→本為警。諸跨域見涉框間譯，譯恒失物。失不可見→未得，非避。深察域界——隱設居此。常隱處：於各域工作異之共喻、設為跨域因之統關、結構持而量不持之喻。

## 驗

- [ ] 眾已明識並其需塑達
- [ ] 式依見類與眾擇，非習或便
- [ ] 見以貫整述，非分為域摘
- [ ] 諸貢域為驗而名，非僅歸
- [ ] 諸簡明述——何擱、何近
- [ ] 現值已述——整啟部不啟何
- [ ] 多域詞保，非平為一域語
- [ ] 入點依眾舊識擇——自其所在起，橋至見所往
- [ ] 見或誤之最強由已述
- [ ] 見可證偽——可改之特證或論已名
- [ ] 域專讀其域貢可認為正，非漫

## 忌

- **逐域報**：逐示各域貢非達見——乃示原料。見乃自合現。先以合領，需則後以域細撐
- **收斂偽確**：三域皆指同方覺如強證。然若諸域共底設或數源→收斂較顯獨立為少。恒察諸域真獨否
- **平至眾域**：達專時欲全譯見於其語。使可通而破多域性。保質——不熟詞非噪，乃信
- **略挑步**：漏「此以或誤」覺強見。不。反使見不可信且不可改。認誠為特，非弱
- **見漲**：稱合示過實。跨域察非自動為突破。範宜精：「此適 X 於境 Y」勝「此改一切」
- **早達**：gestalt 未全成而達生半見——似整而審下崩。達屢停→議於上之 `integrate-gestalt`，非此
- **藏於繁**：用多域詞以顯精而非保真質。若簡框捕同見而不失關→用簡。繁宜必，非演

## 參

- `integrate-gestalt` — 產此技所達之見；express-insight 為 synoptic 週之達階
- `argumentation` — 為言建邏案；express-insight 達感。Argumentation 言「X 何以真」；express-insight 言「同視 A、B、C 時何現」
- `teach` — 傳已知立識；express-insight 傳初成之現解。教傳；達現
- `shine` — 傳真在於達；express-insight 可用彼光攜多域感而不失暖誠
- `expand-awareness` — 擴使整可能之感場；express-insight 經達所擴場所示，閉週
- `adaptic` — 合全 synoptic 週之元技；express-insight 為清-開-感-整-達序之五末步
