# 教（導）

導人成善師、釋者、講者。AI 為教練——助察何傳於誰、構容、練釋、依饋精、支交、思何成。

## 用

- 人需講技內於眾欲善備→用
- 欲書善文、教程、釋→用
- 釋概於異專級難→用
- 導友或新工欲更效→用
- 備講、坊、知分會→用
- `learn-guidance` 助取知後、今需傳之→用

## 入

- **必**：所欲教（題、概、系、流）
- **必**：眾為誰（專級、境、與人關）
- **可**：交式（講、文、單對、坊）
- **可**：時限（5 分釋、30 分講、書文）
- **可**：前教試與何不行
- **可**：人於題之自舒級（深專對新學）

## 行

### 一：察——解教戰

構容前解教境之全。

1. 問所教與何故：「何概當中、不中則何發？」
2. 識眾：「將釋於誰？已知何？」
3. 察人自解：知題深足以教乎？（否則先薦 `learn-guidance`）
4. 識式：講、文、會、碼覆、結對
5. 定成則：「何知眾解？」
6. 顯憂或慮：「此何部最致緊？」

敗：人不能述眾→助建型：「想一具人聞之。何知？關何？」不能述題→或需先深學。

### 二：構——組容為明

助人建明敘構釋。

1. 識單核訊：「眾若唯憶一事、當為何？」
2. 由核外建：核前何境、後何細？
3. 施倒金字塔：要訊先、支細後
4. 技容擇構模：
   - **概釋**：何 → 何故 → 何如 → 例 → 邊例
   - **教程**：標 → 前提 → 步 → 驗 → 次步
   - **構覽**：問 → 限 → 解 → 衡 → 慮選
   - **調走**：症 → 察 → 根因 → 修 → 防
5. 確各段有明意：段不利核訊→刪
6. 計轉：「我們已論 X。今、建於彼、需解 Y、因...」

得：構綱、各元利核訊。構當覺邏與必——各段自然引次。

敗：構長→範過廣——助刪。構平（皆同級）→序需作——識主從。拒構（「我自然釋」）→記自然釋利簡題敗繁——構為架。

### 三：練——演釋

請人釋概、AI 為眾。

1. 請其釋如於真眾
2. 首遍勿打斷——任覓自然流
3. 記何處明、何處惑模
4. 記何處用眾或不知之術
5. 記何處跳步或設眾或無之知
6. 記何處於易部過久、難部速
7. 若有時限→計時

得：首釋顯人自然教模——可建之長與當調之習。練當覺低注：「此為粗稿、非演。」

敗：人凍或言「不知何始」→歸步二構而請其一段一段釋而非全。過自批（「彼差矣」）→引至具：「實、汝釋 X 之法甚明——焦於使 Y 配此質。」

### 四：精——依饋改

予具、可施饋於練釋。

1. 先長：「汝以 Y 類釋 X 之部甚效、因...」
2. 識最大改機（非諸患——焦一二）
3. 薦具替：「勿言 [繁版]、試：[簡版]」
4. 察知咒：何處其專使之跳眾所需步？
5. 察眾配：深於眾否？淺/深？
6. 用類→察類準（誤類劣於無類）
7. 請再釋精段試改

得：定饋使釋可量改。人覺首二試之差。饋構建——何作、非唯何避。

敗：人於饋防→改「此不明」為「眾或不隨此——如何更明？」精版不勝→問或構（步二）非表——歸綱。

### 五：交——教中支

教即時→交中支。

1. 活講：助前備可能問之答
2. 文：覆書版於明、構、眾配
3. 助備「不知」之刻：「若問汝不能答、言：『好問——將察而後答。』恆可。」
4. 鼓交：助備察問予眾
5. 備復計：眾迷、倦、或越釋時何作
6. 交中導→予簡具促（「此處慢」「彼似惑——察」）

得：人覺備而支。有可能問之答、不期境之策、信不知諸事可。

敗：憂為主阻→直理：備減憂、認緊於眾常生接。式不定→助受式而適、勿控境。

### 六：思——析何成

教事後、導思以續改。

1. 問：「何成？汝以何為傲？」
2. 問：「何處察眾最入、最不入？」
3. 問：「眾之應何驚？」
4. 問：「若可改一事、為何？」
5. 接思於則：「成之部用 [技]。可廣施。」
6. 識下次一具改標
7. 慶成：教為練增之技

得：人得具見於教效——非泛感、為何成何故之具察。離以下次一可施改。

敗：唯見負→引至成具刻。唯見正→輕探眾惑域。無思（即進）→記思為最久改處——縱 5 分覆亦重。

## 驗

- [ ] 構前察教戰（眾、式、限）
- [ ] 識核訊、構繞之
- [ ] 人於交前至少練釋一次
- [ ] 饋具、可施、致可量改
- [ ] 人備問、不確、眾適
- [ ] 後交思識至少一具改於下
- [ ] 導全程鼓——教難當認

## 忌

- **導容、非導教**：助學材而非助講之。需學→先用 `learn-guidance`
- **過構**：使構過剛、人自然教聲失。構當支其式、非代之
- **完美陷**：練無止而不交。某點練收益減——推交
- **忽眾異**：混眾需層釋——核思予諸、細予專、類予新
- **饋過載**：一時予諸記淹。焦一二最大影改
- **忽情備**：教憂實。理信如理容
- **教焦慮為實。理信如理容並重**
