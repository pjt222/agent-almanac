# 浄水

野外水源からの浄水により、安全な飲料水を確保する。

## 使用タイミング

- 処理済み水が入手できない自然環境で飲料水が必要な場合
- 水質不明の水源しか利用できない場合（小川、川、湖、池）
- 脱水リスクのある緊急サバイバル状況
- 調理や傷の洗浄のために安全な水が必要な場合

## 入力

- **必須**: 水源（流水または静水）
- **必須**: 容器（金属ポット、ボトル、または即席の容器）
- **任意**: 浄水用品（化学タブレット、フィルター、UVペン）
- **任意**: 煮沸用の火起こし能力（`make-fire`参照）
- **任意**: プレフィルタリング用の布または天然フィルター素材

## 手順

### ステップ1: 水源の評価と選択

すべての水源が同じリスクを持つわけではない。最善の水源を選ぶ。

```
水源の優先順位（最良から最悪）：
┌──────┬─────────────────────────┬────────────────────────────────────┐
│ 順位 │ 水源                    │ 備考                               │
├──────┼─────────────────────────┼────────────────────────────────────┤
│ 1    │ 湧き水（源泉）          │ 汚染リスク最低；それでも処理する   │
│ 2    │ 急流の小川              │ 流水は静水より病原体が少ない       │
│      │ （人間活動の上流）      │                                    │
│ 3    │ 大きな川                │ 希釈効果はあるが上流の農業/         │
│      │                         │ 工業が懸念事項                     │
│ 4    │ 大きな湖                │ 岸辺ではなく開放水面から採水       │
│ 5    │ 小さな池や水たまり      │ 病原体・寄生虫リスク高             │
│ 6    │ 停滞した水たまり        │ 最終手段；強力な処理が必要         │
└──────┴─────────────────────────┴────────────────────────────────────┘

警告サイン（可能なら避ける）：
- 近くに動物の死骸がある
- 藻類の大量発生（青緑色の浮遊物）
- 化学物質の臭いや油膜
- 鉱山、農業地帯、集落の下流
- 周囲に植生がない（有毒な土壌を示す可能性）
```

水面下から採水し（表面の膜を避ける）、岸辺から離れた場所で採水する。

**期待結果：** 最善の水源から清澄またはわずかに濁った水を清潔な容器に採取。

**失敗時：** 劣悪な水源しかない場合（停滞、濁り）、積極的なプレフィルタリング（ステップ2）と複数の浄水方法の併用を計画する。水源が見つからない場合は、谷間の緑の植生、動物の足跡が下り坂に向かっている、夜明け/夕暮れ時の昆虫の群れを探し、流水の音に耳を澄ます。

### ステップ2: 沈殿物のプレフィルタリング

浄化前に粒子状物質を除去する。沈殿物は化学処理の効果を低下させ、フィルターを詰まらせる。

```
即席重力フィルター（底に穴のある容器に層状に配置）：

    ┌─────────────────────┐  ← 上部開口：水を注ぐ
    │  草 / 布            │  ← 粗フィルター
    │  細かい砂            │  ← 微粒子を除去
    │  炭（砕いたもの）    │  ← 一部の化学物質と臭いを吸着
    │  砂利                │  ← 構造的支持
    │  草 / 布            │  ← 砂利の落下を防止
    └────────┬────────────┘
             │
        ろ過された水が滴り落ちる

素材：
- 容器：白樺樹皮の円錐、中空の丸太、カットしたペットボトル、靴下
- 砂：細かい清潔な砂（可能なら先にすすぐ）
- 炭：以前の焚き火からの炭（白い灰ではなく炭のみ）
- 砂利：小さな石、すすいだもの
```

簡単な沈殿物除去には、バンダナ、Tシャツ、または複数層の布で水をこす。

**期待結果：** 目に見えて透明度が増した水。炭層が臭いと味の一部を除去。

**失敗時：** ろ過後もまだ非常に濁っている場合は、容器に入れて30-60分静置する。上層の澄んだ水を慎重に別の容器に移す。静置またはろ過のプロセスを繰り返す。注意：プレフィルタリングだけでは水は安全に飲めない — 浄化の準備段階。

### ステップ4: 水の煮沸

最も信頼性の高いフィールド浄水法。すべての病原体クラスを殺滅する。

```
煮沸手順：
1. 水をローリングボイル（表面を割る大きな泡）にする
2. ローリングボイルを以下の時間維持する：
   - 海抜0m〜2000m / 6500ft：  1分
   - 2000m〜4000m / 6500〜13000ft：  3分
   - 4000m / 13000ft以上：   5分
3. 火から下ろす
4. 蓋をした容器で冷ます
5. 味が平坦な場合、2つの容器の間で数回注ぎ合わせて空気を混入させる

高度補正：
  高度が上がると水は低い温度で沸騰する。
  3000m / 10000ftでは、水は約90°C / 194°Fで沸騰する。
  煮沸時間を延長して低い温度を補う。

燃料の見積もり：
  1Lの煮沸には、容器、風、開始温度に応じて
  約15-20分の持続的な火が必要。
```

**期待結果：** 水が激しいローリングボイルに達し、高度に応じた適切な時間維持される。冷却後、水は生物学的病原体から安全。

**失敗時：** ローリングボイルを維持できない場合（風、弱い火）、煮沸時間を延長する。容器が漏れたり割れたりした場合は別の容器に移す。金属容器がない場合、木製、樹皮製、または革製の容器に熱い石を使って水を沸かせる：石を20分以上火で加熱し、トングや棒で水容器に移す。川の石は避ける（閉じ込められた水分で割れたり爆発する可能性がある）。

### ステップ6: 安全な貯水

浄化された水は汚れた容器や手を通じて再汚染される可能性がある。

```
安全な貯水の実践：
1. 清潔な専用容器に保存する（未浄化容器を再利用しない）
2. 容器を再利用する場合、まず少量の浄化水ですすぐ
3. 容器を密閉または覆う
4. 「未処理」と「浄化済み」の容器を区別またはマークする
   （例：浄化ボトルのパラコードハンドルに結び目をつける）
5. 手を容器の中に入れない — 浸すのではなく注ぐ
6. 暖かい天候では24時間以内に消費する
7. 24時間以上保存した水は再処理する

水分補給計画：
- 最低：2L / 0.5ガロン/日（安静時、涼しい天候）
- 活動時：4-6L / 1-1.5ガロン/日（ハイキング、暑い天候）
- 1日の必要量を満たすよう浄水能力を計画する
```

**期待結果：** 浄化された水が清潔で密閉された容器内で安全に保たれ、未処理水と処理済み水の交差汚染を防ぐシステムが整っている。

**失敗時：** 容器が限られている場合、1つを「未処理」（採水専用）、もう1つを「清潔」（浄化済み専用）に指定する。明確に傷をつけるかマークする。再汚染が疑われる場合、飲む前に再処理する。

## バリデーション

- [ ] 水源を評価し、最善の選択肢を選んだ
- [ ] 濁り水は浄化前にプレフィルタリングした
- [ ] 利用可能なツールと条件に適した浄水方法を選んだ
- [ ] 煮沸は高度補正した時間でローリングボイルに達し維持した
- [ ] 化学処理は正しい投与量と待機時間を使用した
- [ ] 浄化水は清潔で密閉されたラベル付き容器に保存した
- [ ] 1日の水分補給必要量を満たす十分な水を浄化した

## よくある落とし穴

- **プレフィルタリングの省略**: 沈殿物は化学効果を低下させ、フィルターを詰まらせる。濁り水は必ずプレフィルタリングする
- **不完全な煮沸**: 底にいくつかの気泡があるだけではローリングボイルではない。激しい表面を割る気泡を待つ
- **高度の無視**: 高度では水がより低い温度で沸騰する。煮沸時間を適切に延長する
- **化学物質の過少投与**: 冷水や濁り水はより多くの化学物質またはより長い接触時間が必要
- **交差汚染**: 未処理水と浄化水に同じ容器を使用したり、汚れた手で飲み口を触る
- **最悪の水源に単一方法のみ**: 停滞水や家畜隣接水には2つの方法を使用する（フィルター+化学処理、または煮沸+化学処理）
